Rotina Sesta Bebé

⭐⭐ Intermédio 🍼 Bebé ⏱️ 5-7 minutos

Cria o ambiente perfeito para a sesta do bebé: estores fecham, luz suave acende, som branco toca e a temperatura ajusta-se. Tudo com um toque ou automaticamente à hora da sesta.

📦 Equipamentos Necessários

  • 🪟 Estores motorizados Tuya Ou tomada para estores
  • 💡 Lâmpada inteligente No quarto do bebé
  • 🔊 Altifalante inteligente Para som branco
  • 🌡️ Sensor de temperatura Opcional
  • 🔘 Botão inteligente (opcional) Para iniciar rotina manualmente

🌙 Benefícios de uma rotina consistente

Bebés respondem bem a rotinas. Ao repetires os mesmos passos sempre antes da sesta, o bebé associa essas ações a "está na hora de dormir" e adormece mais facilmente.

📱 Configuração Passo-a-Passo

Passo 1: Criar cena "Sesta Bebé" no SmartLife

Vai a "Smart" → "Scene" → "+" e cria uma cena com nome "Sesta Bebé":

  • Ação 1: Estores → "Fechar" (se tiveres estores motorizados)
  • Ação 2: Lâmpada → "Ligar" com intensidade 5% (cor vermelha suave)
  • Ação 3: Tomada do altifalante → "Ligar" (para tocar som branco)

Passo 2: Configurar som branco

Para o som branco, tens duas opções:

  • Altifalante Bluetooth: LIGA na tomada inteligente. O altifalante já deve estar configurado para tocar som branco ao ligar.
  • Alexa/Google: No IFTTT, podes enviar comando "Alexa, toca som branco"

Passo 3: Configurar temperatura ideal

Se tiveres sensor de temperatura e AC/aquecedor:

  • Na mesma cena, adiciona "Then" para ajustar termostato a 21°C
  • Ou cria uma automação separada que verifica temperatura antes da sesta

Passo 4: Criar trigger automático por horário

Para ativar automaticamente à hora da sesta:

  • Vai a "Automation" → "Smart" → "+"
  • If: "Schedule" → "Every day at 13:30" (ajusta conforme a rotina do bebé)
  • Then: "Ativar cena" → "Sesta Bebé"

Passo 5: Criar trigger manual com botão

Para iniciares a sesta manualmente com um botão:

  • If: Botão inteligente → "Single click"
  • Then: "Ativar cena" → "Sesta Bebé"

Passo 6: Criar automação para acordar

Para quando o bebé acorda:

  • Cria outra cena "Acordar Bebé"
  • Ações: Abrir estores (50%), luz desliga, som desliga
  • Podes ativar manualmente ou com sensor de movimento no berço

💡 Dicas e Variações

⏱️ Rotina gradual

Em vez de tudo ao mesmo tempo, cria uma sequência:

  • Minuto 0: Estores fecham (50%) - sinal que a sesta está perto
  • Minuto 5: Som branco começa baixinho
  • Minuto 10: Estores fecham completamente, luz suave acende

📱 Notificação para os pais

Adiciona uma notificação: "😴 Hora da sesta do bebé! Ambiente preparado." Útil se estiveres noutra divisão.

🌡️ Ajuste conforme estação

Podes criar cenas diferentes para verão e inverno:

  • Verão: Estores fecham + AC liga
  • Inverno: Estores fecham + aquecedor liga

🎵 Diferentes sons para diferentes sestas

Manhã: Som mais alegre (pássaros). Tarde: Som branco. Noite: Som de chuva suave.

⚠️ Problemas Comuns e Soluções

Bebé acorda com o barulho dos estores

Solução: Fecha os estores antes de o bebé estar a dormir. Ou usa cortinas elétricas mais silenciosas.

Som branco não começa

Solução: Verifica se o altifalante está configurado para tocar automaticamente ao ligar. Podes precisar de um altifalante com memória.

Horário da sesta muda todos os dias

Solução: Em vez de horário fixo, usa um botão. Quando vês sinais de sono, carregas no botão e a rotina começa.
