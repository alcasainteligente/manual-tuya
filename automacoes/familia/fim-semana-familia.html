Fim de Semana em Família

⭐⭐ Intermédio 👪 Família ⏱️ 5-7 minutos

Começa o fim de semana com uma mensagem personalizada que inclui sugestões de atividades baseadas no clima. Se estiver sol, sugere ida à praia; se estiver chuva, sugere filmes ou jogos de tabuleiro.

📦 Equipamentos Necessários

  • 🔊 Altifalante inteligente Alexa, Google Home
  • 📱 Conta IFTTT Grátis
  • ☁️ Serviço de meteorologia Weather Underground
  • 📅 Google Calendar (opcional) Para eventos familiares

📱 Configuração Passo-a-Passo

Passo 1: Criar conta no Weather Underground

Acede a wunderground.com:

  • Cria uma conta gratuita
  • Obtém uma chave de API
  • No IFTTT, conecta o serviço com a chave

Passo 2: Criar Applet para sábado de manhã

No IFTTT, cria um applet:

  • If: "Date & Time" → "Every day at 08:00"
  • E adiciona condição: apenas sábado
  • Then: "Weather Underground" → "Get current weather"

Passo 3: Criar mensagem personalizada

Usa "Filter code" para criar uma mensagem conforme o clima:

// Filter code JavaScript let weather = Weather.currentCondition; let temp = Weather.currentTemp; let message = "Bom dia família! Hoje é sábado."; if (weather.indexOf('sun') >= 0 || weather.indexOf('clear') >= 0) { message += " Está um dia de sol! ☀️ Sugestão: praia, piquenique ou passeio no parque."; } else if (weather.indexOf('rain') >= 0) { message += " Está a chover. ☔️ Sugestão: filme em família, jogos de tabuleiro ou fazer bolachas."; } else if (temp < 15) { message += " Está frio. ❄️ Sugestão: museu, cinema ou tarde de chocolate quente."; } else { message += " Sugestão: o que acham de um passeio?"; } message += " Bom fim de semana!";

Passo 4: Enviar mensagem

Then: "Alexa" → "Say a phrase" com a mensagem personalizada

Ou envia notificação para o telemóvel de todos.

Passo 5: Integrar com calendário

Se tiveres eventos no Google Calendar, podes incluí-los:

  • "Hoje às 15h, temos o jogo do [nome]. Não se esqueçam!"
  • "Aniversário da avó amanhã. Já compraram presente?"

💡 Dicas e Variações

🍳 Pequeno-almoço especial

No sábado de manhã, em vez da rotina normal, podes ter um pequeno-almoço diferente. Por exemplo, "Hoje é dia de panquecas!" e a luz da cozinha muda para uma cor especial.

🗳️ Votação em família

Quando a sugestão é dada, podes criar um sistema de votação. Cada membro vota no que prefere fazer e a atividade com mais votos ganha.

📸 Registar os fins de semana

No final do dia, tira uma foto em família. Cria um álbum digital "Fins de semana em família".

🌙 Domingo à noite

Cria uma mensagem para domingo à noite: "Espero que tenham tido um ótimo fim de semana. Amanhã é segunda, vamos preparar as mochilas e deitar cedo."

⚠️ Problemas Comuns e Soluções

Sugestões nem sempre agradam a todos

Solução: Rodízio de escolhas. Um fim de semana um escolhe, no outro outro. A sugestão é apenas uma ideia, não uma ordem.

Mensagem muito cedo (fim de semana é para dormir)

Solução: Altera a hora para 10:00 no sábado e domingo. Ou cria duas versões: uma para quem gosta de acordar cedo, outra para tarde.

Previsão do tempo errada

Solução: Usa a previsão do dia em vez da condição atual. Ou aceita que, se o tempo mudar, a família é flexível.
