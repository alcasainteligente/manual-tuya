Alerta Fralda Molhada

⭐⭐⭐ Avançado 🍼 Bebé ⏱️ 6-8 minutos

Recebe uma notificação no telemóvel sempre que a fralda do bebé está molhada. Ideal para prevenir assaduras e garantir o conforto do bebé, especialmente durante a noite.

📦 Equipamentos Necessários

  • 💧 Sensor de humidade Tuya Pequeno e à prova de água
  • 📱 App SmartLife No telemóvel
  • 🔋 Baterias de botão Para o sensor
  • 🩲 Fita adesiva ou bolso para fralda Para fixar o sensor

⚠️ Importante Saber

Segurança em primeiro lugar!

O sensor deve ser à prova de água e colocado numa bolsa ou compartimento seguro, nunca em contacto direto com a pele do bebé. Certifica-te que não tem bordas afiadas.

Existem sensores específicos para fralda no mercado. Se usares um sensor genérico, coloca-o dentro de uma pequena bolsa de tecido.

📱 Configuração Passo-a-Passo

Passo 1: Preparar o sensor

Antes de usar com o bebé:

  • Testa o sensor num copo com água para verificar se deteta humidade
  • Coloca o sensor dentro de uma pequena bolsa de tecido (tipo saquinho de organza)
  • Costura ou fecha bem para não sair
  • Prende a bolsa à fralda com um alfinete de segurança (com cuidado)

Passo 2: Adicionar sensor ao SmartLife

No app:

  • Adiciona o sensor como "Sensor de humidade" ou "Water leak sensor"
  • Dá-lhe um nome: "Sensor Fralda Bebé"
  • Coloca-o na divisão "Quarto do Bebé"

Passo 3: Configurar automação

Cria uma automação no SmartLife:

  • Vai a "Automation" → "Smart" → "+"
  • If: "Sensor Fralda Bebé" → "Water detected" (ou humidade detetada)
  • Then: "Send notification" → "🚼 Fralda do bebé molhada! Hora de trocar."

Passo 4: Adicionar notificação sonora (opcional)

Para não perderes a notificação:

  • Then (outra ação): "Ligar tomada inteligente" onde está um altifalante
  • O altifalante toca um som suave (podes usar um MP3 player com ficha inteligente)

Passo 5: Configurar alerta noturno

Durante a noite, podes querer um alerta diferente:

  • Cria uma segunda automação com horário
  • If: "Sensor Fralda Bebé" → "Water detected"
  • And: Horário entre 22h e 8h
  • Then: "Send notification" com tom de alerta mais alto

💡 Dicas e Variações

🔄 Reutilizar o sensor

O sensor pode ser reutilizado. Depois de usar, limpa com álcool e deixa secar completamente antes de voltar a usar.

📊 Registo de trocas

Combina com a automação "Registo em Planilha" para guardar histórico de trocas de fralda. Útil para perceber padrões e para mostrar ao pediatra.

🌙 Versão com luz noturna

Em vez de notificação sonora, podes configurar para acender uma luz vermelha suave no quarto dos pais. Menos perturbador durante a noite.

📱 Aplicações específicas

Existem sensores próprios para fralda (ex: MonBaby, Lully). Alguns já têm app própria e não precisam de SmartLife.

⚠️ Problemas Comuns e Soluções

Falsos positivos (suor, humidade do ar)

Solução:

  • Ajusta a sensibilidade do sensor (se disponível)
  • Coloca o sensor numa bolsa mais isolada
  • Adiciona um delay de 30 segundos antes do alerta (confirma que é mesmo xixi)

Sensor não deteta xixi

Solução:

  • Verifica se o sensor está bem posicionado na zona da fralda que molha primeiro
  • Testa com água para confirmar que funciona
  • Pode ser necessário usar dois sensores (um de cada lado)

Bebé desconfortável com o sensor

Solução: Usa uma bolsa de tecido muito macio e coloca o sensor na parte de fora da fralda, entre a fralda e a roupa, em vez de dentro.
