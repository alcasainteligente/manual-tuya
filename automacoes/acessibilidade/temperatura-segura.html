Temperatura Segura

⭐ Iniciante 😊 Conforto ⏱️ 4-6 minutos 🌡️ Prevenção de riscos

Garante que a temperatura da casa se mantém sempre dentro de limites seguros para pessoas idosas (18°C - 25°C). Evita hipotermia no inverno e golpes de calor no verão, mesmo quando não estás presente.

📦 Equipamentos Necessários

  • 🌡️ Termostato inteligente Com controlo de temperatura
  • 🌡️ Sensor de temperatura (opcional) Para monitorização
  • ❄️ AC ou aquecedor inteligente Para ajustar temperatura

📱 Configuração Passo-a-Passo

Passo 1: Definir limites de temperatura segura

Para pessoas idosas, recomenda-se:

  • Temperatura mínima: 18°C (risco de hipotermia abaixo disto)
  • Temperatura máxima: 25°C (risco de golpe de calor acima disto)
  • Temperatura ideal: 20-22°C

Passo 2: Criar automação para frio

IF: "Sensor Temperatura" < 18°C

THEN: "Aquecimento" = ON, temperatura 20°C

THEN: Notificação: "❄️ Temperatura baixa detetada. Aquecimento ligado para segurança."

Nome: "Proteção Contra Frio"

Passo 3: Criar automação para calor

IF: "Sensor Temperatura" > 25°C

THEN: "AC" = ON, temperatura 23°C (modo arrefecimento)

THEN: Notificação: "☀️ Temperatura alta detetada. AC ligado para segurança."

Nome: "Proteção Contra Calor"

Passo 4: Desligar automático

IF: "Sensor Temperatura" entre 19°C e 24°C (zona segura)

THEN: "Aquecimento/AC" = OFF

Passo 5: Alerta para familiares

Se a temperatura não normalizar após 30 minutos de climatização:

IF: Temperatura < 18°C por 30 minutos + Aquecimento ON

THEN: Notificação para familiar: "⚠️ A temperatura continua baixa. Possível avaria no aquecimento."

Passo 6: Versão com horário noturno

Durante a noite, a temperatura pode ser ligeiramente mais baixa (17°C) para melhor sono, desde que a pessoa tenha cobertores adequados.

💡 Dicas e Variações

👵 Variação 1: Sensibilidade individual

Algumas pessoas idosas sentem mais frio. Ajusta os limites conforme a necessidade:

  • Mínimo: 20°C (para quem sente muito frio)
  • Máximo: 24°C (para quem não tolera calor)

🏠 Variação 2: Zonas diferentes

Podes ter temperaturas diferentes por divisão:

  • Quarto: 18°C (melhor para dormir)
  • Sala: 21°C (conforto durante o dia)

🌡️ Variação 3: Monitorização remota

Recebe um relatório diário da temperatura da casa, mesmo que esteja dentro dos limites.

🔌 Variação 4: Backup de emergência

Se o aquecimento falhar, podes ter um aquecedor elétrico de reserva que liga automaticamente.

📱 Variação 5: Controlo por voz

A pessoa pode pedir: "Alexa, está muito frio" e o sistema aumenta a temperatura.

⚠️ Problemas Comuns e Soluções

Pessoa idosa não sente frio/calor (perda de sensibilidade)

Solução: Esta automação é ainda mais importante neste caso. Mantém os limites apertados.

AC/aquecimento avariado

Solução: O alerta de temperatura anormal persistente notifica os familiares para intervirem.

Pessoa abre janelas no inverno (quer arejar)

Solução: Usa sensor de janela. Se abrir, desliga o aquecimento temporariamente e reativa quando fechar.

Conta de luz elevada

Solução: Usa mantas elétricas ou aquecedores locais em vez de aquecer a casa toda.
