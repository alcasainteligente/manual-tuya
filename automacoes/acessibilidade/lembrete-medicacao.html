Lembrete Medicação

⭐ Iniciante 💊 Medicação ⏱️ 4-6 minutos ⏰ Horário dos medicamentos

Nunca mais esquecer a medicação. O sistema emite alertas sonoros e visuais nos horários programados, com informações sobre o medicamento e dosagem. Ideal para idosos ou pessoas com múltiplos medicamentos.

📦 Equipamentos Necessários

  • 🎵 Coluna inteligente Para alertas sonoros
  • 💡 Lâmpada inteligente Para alerta visual
  • 📱 Telemóvel Para notificações
  • 📋 App SmartLife Para programar horários

📱 Configuração Passo-a-Passo

Passo 1: Criar automações para cada medicamento

Vai a "Automation" → "+".

IF: "Timing" → 08:00, 13:00, 20:00 (horários da medicação)

THEN: "Coluna" = Tocar mensagem: "Está na hora de tomar [medicamento] - [dosagem]"

Nome: "Lembrete Medicação Manhã"

Passo 2: Adicionar alerta visual

THEN: "Luz da sala" = Piscar 3 vezes (padrão de alerta)

Ou mudar para cor vermelha durante 5 minutos.

Passo 3: Notificação no telemóvel

THEN: Notificação: "💊 Hora do medicamento: [nome] - [dosagem]"

Passo 4: Repetir se não for confirmado

Delay: 15 minutos

IF: Não confirmado (sem sensor ou botão)

THEN: Repetir alerta + Notificar familiar

Passo 5: Botão de confirmação

Cria um botão virtual "Tomei o remédio". Quando pressionado, cancela os alertas.

Passo 6: Versão com sensor na caixa

Se tiveres um sensor na caixa de medicamentos, podes detetar quando é aberta e não precisar de confirmação manual.

💡 Dicas e Variações

📋 Variação 1: Lista completa de medicamentos

Cria uma tabela com todos os medicamentos, horários e dosagens. Podes ter uma automação para cada.

  • 08:00: Aspirina 100mg + Omeprazol 20mg
  • 13:00: Metformina 500mg
  • 20:00: Sinvastatina 40mg

👥 Variação 2: Notificação para cuidadores

Se a pessoa tem dificuldade em lembrar-se, envia também notificação para o cuidador/familiar.

📊 Variação 3: Histórico de medicação

Regista quantas vezes a medicação foi tomada. Útil para consultas médicas.

🔔 Variação 4: Diferentes sons para diferentes medicamentos

Usa sons diferentes para cada medicamento. A pessoa associa o som ao remédio.

🏥 Variação 5: Alerta de reposição

Quando a medicação está a acabar (por contagem de dias), recebe alerta para comprar mais.

⚠️ Problemas Comuns e Soluções

Pessoa ignora os alertas

Solução: Aumenta a frequência ou envolve um familiar para ligar a confirmar.

Muitos medicamentos, confunde-se

Solução: Usa um dispensador automático (próxima automação) em vez de apenas lembretes.

Alerta toca mas pessoa está fora de casa

Solução: Envia notificação para o telemóvel também. Pode tomar a medicação onde estiver.

Mudança de horários (fim de semana)

Solução: Cria automações diferentes para dias de semana e fim de semana.
