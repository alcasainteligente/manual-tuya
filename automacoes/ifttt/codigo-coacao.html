Código de Coação

⭐⭐⭐ Avançado 🔒 Segurança ⏱️ 7-9 minutos

Se fores forçado a desativar o alarme sob coação, um código especial envia um alerta silencioso para a polícia ou familiar, sem que o agressor perceba.

📦 Equipamentos Necessários

  • 🔒 Central de alarme Tuya Com código de coação
  • 📱 Conta IFTTT Grátis ou Pro
  • 📨 Serviço de SMS (Twilio) Para alerta silencioso
  • 📞 Número de emergência Familiar ou polícia

⚠️ INFORMAÇÃO DE SEGURANÇA

Este sistema pode salvar vidas, mas requer configuração cuidadosa.

O código de coação é um código alternativo que parece desativar o alarme normalmente, mas na verdade envia um alerta silencioso. O agressor não percebe que o alerta foi enviado.

Combina com as autoridades locais para saber qual o procedimento correto na tua área.

📱 Configuração Passo-a-Passo

Passo 1: Verificar se o alarme suporta código de coação

Nem todos os alarmes Tuya têm esta funcionalidade. Verifica no manual ou app:

  • Procura por "Duress code" ou "Código de coação"
  • Alguns modelos permitem configurar um código alternativo
  • Se não tiveres, podes simular com um botão secreto

Passo 2: Configurar código de coação no alarme

No app SmartLife:

  • Vai às definições do alarme
  • Procura por "Advanced settings" ou "Duress code"
  • Define um código diferente do normal (ex: código normal 1234, coação 4321)
  • Guarda as alterações
📌 Escolhe um código fácil de lembrar mas diferente do habitual. Ex: se o teu código é 1234, usa 1243 para coação.

Passo 3: Configurar IFTTT para detetar o código de coação

No IFTTT, cria um novo Applet:

  • If This: Smart Life → Alarm system
  • Seleciona "Disarmed with duress code" ou equivalente
  • Se não existir essa opção, usa "Disarmed" + filtro por código

Passo 4: Configurar alerta silencioso

Agora configura a ação que deve acontecer:

  • Then That: Twilio → Send SMS
  • Configura a mensagem para um número de confiança
To: +351911222333
Body:
🚨 ALERTA DE COAÇÃO - CÓDIGO SILENCIOSO 🚨

O código de coação foi usado em casa!
Data/Hora: {{OccurredAt}}
Local: {{DeviceLocation}}

A pessoa pode estar em perigo.
Por favor verifique com discrição.

NÃO uses palavras como "polícia" ou "emergência" na mensagem, caso o agressor veja o telemóvel.

Passo 5: Configurar ações adicionais (opcional)

Podes adicionar mais ações para aumentar a segurança:

  • 📸 Ativar câmara para gravar (sem flash, silenciosamente)
  • 🔊 Ativar altifalante com som de "tudo normal" (para enganar)
  • 💡 Ligar luzes como se nada tivesse acontecido
  • 📱 Enviar notificação para vizinho de confiança

Passo 6: Testar o sistema com cuidado

Testa numa situação controlada:

  • Informa o contacto que vais testar (para não alarmar)
  • Usa o código de coação e verifica se recebe SMS
  • Confirma que as ações adicionais funcionam
  • Pratica com a família para todos saberem o código

💡 Dicas e Variações

🤫 Alternativa sem código de coação

Se o teu alarme não suportar, podes simular com um botão secreto:

  • Esconde um botão inteligente num local discreto
  • Configura no IFTTT: If button pressed → Then send SMS
  • Garante que o botão parece inofensivo (ex: dentro de uma gaveta)

📱 Usar palavras-código no telemóvel

Podes configurar no IFTTT para enviar SMS se enviares uma palavra específica por SMS:

  • Se enviares "Está tudo bem" para um número, significa que estás em perigo
  • O IFTTT deteta a SMS e aciona as ações

🔄 Cadeia de contactos

Configura uma sequência para garantir que alguém recebe o alerta:

  • 1º Envia SMS para familiar
  • Se não for lido em 2min, envia para outro
  • Por fim, envia email para todos

🎭 Disfarçar as ações

Para não levantar suspeitas, as ações devem parecer normais:

  • A câmara grava mas o LED não acende
  • As luzes acendem como se fosse rotina normal
  • O altifalante diz "Alarme desativado. Modo noturno ativo."

📍 Incluir localização em tempo real

Se possível, envia a localização GPS do telemóvel da pessoa. Pode ser crucial para saber se foi levada de casa.

⚠️ Problemas Comuns e Soluções

O código de coação não é reconhecido

Solução:

  • Verifica no manual se o modelo suporta
  • Testa com diferentes combinações
  • Considera a alternativa do botão escondido

O alerta não é enviado

Solução:

  • Confirma que o IFTTT está ligado ao Twilio
  • Verifica créditos no Twilio
  • Testa com um SMS manual primeiro

O agressor pode ver a mensagem

Solução: Usa palavras-código na mensagem:

  • Em vez de "Alerta de coação", usa "A Maria precisa de ir ao médico"
  • Combina um código com o contacto: "Há bolo no forno" = perigo
