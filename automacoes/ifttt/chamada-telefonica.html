Chamada Telefónica

⭐⭐⭐ Avançado 📞 Chamada ⏱️ 7-9 minutos

Recebe uma chamada automática em caso de emergência. A chamada repete uma mensagem gravada até atenderes ou deixar recado na voicemail.

📦 Equipamentos Necessários

  • 📱 Qualquer sensor Tuya Emergência (fumo, água, etc.)
  • 📱 Conta IFTTT Pro Necessário para chamadas
  • 📞 Conta Twilio Com capacidade de voz
  • 🎤 Mensagem gravada Ficheiro MP3 ou texto

⚠️ AVISO IMPORTANTE

IFTTT Pro é necessário para ações de chamada telefónica.

A versão gratuita do IFTTT só permite notificações e SMS. Para chamadas de voz, precisas de subscrição Pro (cerca de $3/mês).

📱 Configuração Passo-a-Passo

Passo 1: Configurar Twilio para chamadas

No Twilio, precisas de um número com capacidade de voz:

  • Acede a twilio.com e faz login
  • Compra um número de telefone (custo ~$1/mês)
  • Certifica-te que o número suporta voz
  • Guarda o Account SID e Auth Token

Passo 2: Gravar mensagem de voz

Podes gravar uma mensagem ou usar texto-para-voz:

"Alerta de emergência na sua casa! O sensor de fumo foi ativado. Por favor verifique imediatamente. Esta é uma mensagem automática da AL Casa Inteligente."

Guarda o ficheiro MP3 num URL público (Dropbox, Google Drive).

Passo 3: Configurar TwiML (linguagem de chamadas)

No Twilio Console, cria um TwiML Bin com:

<Response> <Say voice="alice" language="pt-PT"> Alerta de emergência na sua casa! O sensor de fumo foi ativado. Por favor verifique imediatamente. </Say> <Pause length="2"/> <Say> Esta mensagem vai repetir-se. Se for uma emergência real, ligue 112. </Say> </Response>

Passo 4: Atualizar para IFTTT Pro

Se ainda não tens IFTTT Pro:

  • Vai a ifttt.com/pro
  • Escolhe o plano mensal ou anual
  • Ativa a subscrição

Passo 5: Criar Applet no IFTTT Pro

Cria um novo Applet:

  • If This: Smart Life → Escolhe o sensor de emergência
  • Then That: Twilio → Make a phone call
  • Configura o número de destino
  • Escolhe a mensagem (URL do TwiML ou texto)

Passo 6: Configurar repetição

Para garantir que a chamada é atendida:

  • Configura para repetir a chamada a cada 5 minutos se não atender
  • Até 3 tentativas
  • Deixar recado na voicemail na última tentativa

💡 Dicas e Variações

📱 Chamada internacional

Se estiveres de férias no estrangeiro, podes configurar para ligar para o teu número internacional. Verifica as tarifas do Twilio.

🔄 Cadeia de chamadas

Configura uma sequência:

  • 1º liga para ti
  • Se não atenderes, liga para familiar
  • Se ninguém atender, liga para vizinho

💰 Alternativa mais barata - SMS

Chamadas têm custo. Se quiseres algo mais económico, usa SMS (mais barato) ou Telegram (grátis).

🔊 Mensagem personalizada por sensor

Cria mensagens diferentes:

  • Fumo: "Incêndio detetado!"
  • Água: "Inundação na casa de banho"
  • Intrusão: "Alarme perimetral ativado"

⚠️ Problemas Comuns e Soluções

A chamada não é feita

Solução:

  • Verifica se tens IFTTT Pro ativo
  • Confirma créditos no Twilio
  • Testa o número manualmente

A mensagem não é percetível

Solução:

  • Usa voz "alice" do Twilio (muito natural)
  • Fala devagar e pausadamente
  • Testa com diferentes vozes

Chamadas são muito caras

Solução:

  • Usa SMS em vez de chamada
  • Compra pacotes de minutos no Twilio
  • Limita a 1 chamada por hora
