Luz com Comando Twitter

⭐⭐ Intermédio 🐦 Twitter ⏱️ 5-7 minutos

Controla as luzes da tua casa através do Twitter. Publica um tweet com #acenderluz e a luz acende. Ideal para gadgets e demonstrações.

📦 Equipamentos Necessários

  • 💡 Lâmpada inteligente Tuya Qualquer modelo
  • 📱 Conta IFTTT Grátis
  • 🐦 Conta Twitter Pode ser privada

⚠️ AVISO DE PRIVACIDADE

Recomenda-se usar uma conta Twitter privada.

Se usares uma conta pública, qualquer pessoa pode controlar as tuas luzes se descobrir as hashtags. Usa uma conta privada e segue-te a ti mesmo.

📱 Configuração Passo-a-Passo

Passo 1: Criar conta Twitter privada (opcional)

Para segurança, cria uma conta secundária:

  • Cria uma nova conta no Twitter
  • Vai a Definições → Privacidade
  • Ativa "Proteger os meus Tweets"
  • Segue-te a ti mesmo da conta principal

Passo 2: Ligar Twitter ao IFTTT

No IFTTT:

  • Vai a "Explore" e pesquisa por "Twitter"
  • Clica em "Connect" e faz login na conta que vais usar
  • Autoriza o acesso

Passo 3: Ligar Tuya ao IFTTT

Se ainda não ligaste:

  • Pesquisa por "Smart Life" ou "Tuya"
  • Conecta com a tua conta

Passo 4: Criar Applet para Acender Luz

Cria um novo Applet:

  • If This: Twitter → New tweet by you with hashtag
  • Configura a hashtag: #acenderluz
  • Then That: Smart Life → Turn on
  • Seleciona a lâmpada que queres controlar

Passo 5: Criar Applet para Apagar Luz

Repete o processo para apagar:

  • If This: Twitter → New tweet by you with hashtag
  • Hashtag: #apagarluz
  • Then That: Smart Life → Turn off

Passo 6: Criar Applet para outras cores (opcional)

Se tiveres lâmpadas RGB:

  • #luzvermelha → mudar para vermelho
  • #luzazul → mudar para azul
  • Precisas de usar cenas (scenes) no SmartLife

Passo 7: Testar

Testa o sistema:

  • Publica um tweet com #acenderluz
  • Espera alguns segundos
  • A luz deve acender
  • Testa #apagarluz
📌 Pode demorar até 1 minuto para o IFTTT detetar o tweet.

💡 Dicas e Variações

🔐 Conta privada + Seguir-te

Configura assim:

  • Conta A (pública) - tua conta normal
  • Conta B (privada) - para comandos
  • Segue a Conta B com a Conta A
  • Publicas tweets da Conta A, mas só a Conta B vê

📝 Múltiplos comandos num tweet

Podes usar várias hashtags no mesmo tweet:

  • "Hoje vou #acenderluz da sala e #apagarluz do quarto"
  • O IFTTT deteta cada hashtag separadamente

🎮 Brincadeiras com amigos

Se confiares, podes autorizar amigos a controlar as luzes. Basta eles usarem as hashtags e seguirem a conta.

📱 Alternativa com SMS

Se não quiseres usar Twitter, podes fazer o mesmo com SMS (ver automação "Comando por SMS").

🤖 Resposta automática

Configura o IFTTT para responder ao teu tweet:

  • Then Another: Twitter → Post a tweet
  • "@teuuser A luz foi acesa às {{OccurredAt}}"

⚠️ Problemas Comuns e Soluções

O IFTTT não deteta o tweet

Solução:

  • Verifica se a conta do Twitter está corretamente ligada
  • Confirma que a hashtag está exata (#acenderluz vs #AcenderLuz)
  • O IFTTT pode demorar até 15 minutos - tem paciência

Alguém está a controlar as minhas luzes

Solução: Muda a conta para privada imediatamente. Depois altera as hashtags para algo mais pessoal (#acenderluz123).

A luz não acende/apaga

Solução:

  • Testa no app SmartLife se a luz responde
  • Verifica se a lâmpada está online
  • Confirma que selecionaste a lâmpada certa no IFTTT
