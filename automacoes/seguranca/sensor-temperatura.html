Sensor de Temperatura Extrema

⭐⭐ Intermédio 📍 Geral ⏱️ 5 minutos

Alerta se a temperatura da casa descer abaixo dos 5° ou subir acima dos 35°. Protege idosos, bebés, animais e canalizações contra temperaturas extremas.

📦 Equipamentos Necessários

  • 🌡️ Sensor de temperatura e humidade Tuya Zigbee ou Wi-Fi
  • 🔋 Baterias AA ou CR2032
  • 📡 Hub Zigbee (se sensor Zigbee) Opcional
  • 🌡️ AC ou Aquecedor inteligente Para ação corretiva

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar os sensores

Coloca sensores de temperatura em locais estratégicos:

  • Quarto de bebé/idoso: Monitorização prioritária
  • Sala: Temperatura ambiente geral
  • Cave: Risco de temperaturas baixas
  • Sótão: Risco de temperaturas altas
  • Exterior: Para comparação
📌 Evitar locais com luz solar direta ou perto de fontes de calor/frio.

Passo 2: Adicionar sensores ao app

Adiciona todos os sensores ao app SmartLife.

Dá nomes: "Sensor Teto Quarto Bebé", "Sensor Temp Sala", etc.

Passo 3: Criar automação para temperatura muito baixa

Vai a "Automation" → "+".

IF: "Device Status" → Sensor Temperatura < 5°C

THEN:

  • Notificação: "❄️ Alerta! Temperatura abaixo dos 5°C em [local]"
  • Ligar aquecedor (se disponível)
  • Notificação para familiar

Nome: "Alerta Frio Extremo"

Passo 4: Criar automação para temperatura muito alta

IF: "Device Status" → Sensor Temperatura > 35°C

THEN:

  • Notificação: "🔥 Alerta! Temperatura acima dos 35°C em [local]"
  • Ligar AC/ventilador
  • Abrir estores (se automáticos)
  • Notificação para familiar

Nome: "Alerta Calor Extremo"

Passo 5: Criar automação para quarto de bebé

IF: Sensor Quarto Bebé < 19°C OU > 23°C

THEN: Notificação: "👶 Temperatura do quarto do bebé fora do ideal: [temperatura]°C"

Nome: "Alerta Temperatura Bebé"

Passo 6: Criar automação para risco de canos congelados

IF: Sensor Cave/Exterior < 2°C

THEN:

  • Notificação: "⚠️ Risco de canos congelados! Ligar aquecimento"
  • Ligar aquecimento mínimo (15°C)

Nome: "Proteção Canos"

💡 Dicas e Variações

👵 Variação 1: Idosos em casa

Para idosos, os limites devem ser mais apertados:

  • Mínimo: 18°C (risco de hipotermia)
  • Máximo: 28°C (risco de desidratação)
  • Notificação para cuidador/familiar

🐱 Variação 2: Animais de estimação

Se deixas animais sozinhos em casa:

  • Limite máximo: 30°C
  • Ligar AC automaticamente
  • Notificação para poderes ver por câmara

🌡️ Variação 3: Temperatura ao longo do dia

Cria um registo diário no IFTTT/Google Sheets:

  • Regista temperatura a cada hora
  • Recebe relatório semanal

🏠 Variação 4: Modo férias

Quando estás de férias:

  • Manter temperatura entre 15°C e 30°C
  • Alerta imediato se sair destes limites
  • Contactar vizinho para verificar

📊 Variação 5: Comparação exterior/interior

Se tiveres sensor exterior:

  • Se exterior < 5° e interior < 15°, alerta de isolamento fraco
  • Sugere melhorias de isolamento

⚠️ Problemas Comuns e Soluções

Leituras incorretas por localização errada

Solução: Certifica-te que os sensores não estão:

  • Perto de janelas (correntes de ar falsificam leituras)
  • Expostos a luz solar direta (lêem temperatura mais alta)
  • Perto de fontes de calor (aquecedores, eletrodomésticos)
  • Em locais com pouca circulação de ar

Baterias gastam-se rapidamente

Solução:

  • Usa pilhas de lítio de qualidade (duram mais)
  • Reduz a frequência de atualização do sensor (se configurável)
  • Verifica se o sensor está dentro do alcance do hub (se estiver longe, gasta mais bateria)

Sensor não conecta ao hub

Solução:

  • Verifica a distância ao hub Zigbee/Wi-Fi
  • Adiciona repetidores de sinal Zigbee (podem ser lâmpadas ou tomadas)
  • Reinicia o sensor e tenta emparelhar novamente

Múltiplos alertas em pouco tempo

Solução: Adiciona um atraso (delay) nas automações:

  • Ex: Se temperatura < 5°, espera 10 minutos antes de enviar novo alerta
  • Evita notificações repetitivas quando a temperatura oscila perto do limite

Sensor não reporta humidade (se aplicável)

Solução: Alguns sensores têm apenas temperatura. Verifica as especificações. Se tiver humidade, pode precisar de calibração.
