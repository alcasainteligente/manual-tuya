Alarme com SMS Emergência

⭐⭐⭐ Avançado 📍 Geral ⏱️ 10 minutos

Envia SMS para contactos de emergência quando qualquer alarme dispara. Garante que és alertado mesmo sem internet ou notificações push. Camada extra de segurança para situações críticas.

📦 Equipamentos Necessários

  • 🔌 Sensores/alarmes Tuya Qualquer dispositivo de segurança
  • 📱 IFTTT (conta gratuita) Para integração com SMS
  • 📲 Serviço de SMS Via IFTTT (Cellular, Telegram, etc.)
  • 📋 Lista de contactos Números de telefone para emergência

💡 Nota: O IFTTT permite enviar SMS gratuitamente até um certo limite. Para uso intensivo, considera serviços pagos ou alternativas como Telegram.

📱 Configuração Passo-a-Passo

Passo 1: Criar conta no IFTTT

Acede a ifttt.com e cria uma conta gratuita.

Instala a app no teu telemóvel para receber notificações.

Passo 2: Ligar os serviços

No IFTTT, pesquisa e liga:

  • SmartLife / Tuya - para os teus dispositivos
  • SMS - ou Telegram, WhatsApp, Email (conforme preferires)
📌 O serviço de SMS pode ter custos associados. O Telegram é gratuito e igualmente eficaz.

Passo 3: Criar applet para cada tipo de alarme

Cria um applet para cada situação:

IF: SmartLife → "Any device triggers" → Seleciona o dispositivo (ex: Sensor de Fumo)

THEN: SMS → "Send SMS" → Insere o número de telefone

Mensagem: "🚨 ALARME: [nome do dispositivo] ativado em [data/hora] - [morada da casa]"

Nome do applet: "Alerta SMS - [nome do dispositivo]"

Passo 4: Configurar múltiplos contactos

Para cada contacto de emergência, cria um applet separado ou usa grupos:

  • Contacto 1: Familiar mais próximo
  • Contacto 2: Vizinho de confiança
  • Contacto 3: Serviço de segurança (se aplicável)

Passo 5: Criar applet com localização (opcional)

Podes incluir a localização da casa na mensagem:

Mensagem: "🚨 ALARME: [dispositivo] ativado - Morada: Rua X, nº Y - [hora]"

Passo 6: Testar

Ativa um sensor manualmente e verifica se o SMS é recebido nos contactos configurados.

Ajusta a mensagem conforme necessário.

💡 Dicas e Variações

📱 Variação 1: Múltiplos canais de notificação

Não te limites ao SMS. Cria applets para:

  • Telegram: Cria um grupo com a família e envia alertas para lá
  • Email: Para registo histórico
  • Chamada telefónica: Alguns serviços IFTTT permitem chamadas automáticas

👨‍👩‍👧 Variação 2: Diferentes contactos para diferentes alarmes

Configura contactos específicos conforme o tipo de alarme:

  • Fumo/Gás: Bombeiros + familiar
  • Intrusão: Polícia + vizinho
  • Queda (idosos): Familiar + emergência médica

🔊 Variação 3: Alerta sonoro + SMS

Combina com outras automações: quando o alarme dispara, ativa sirenes e luzes, e envia SMS.

📸 Variação 4: SMS com foto

Se tiveres câmaras, podes usar serviços como Telegram que permitem enviar fotos. O IFTTT pode enviar a URL da imagem no SMS.

🔄 Variação 5: Confirmação de receção

Podes criar um sistema onde o contacto deve responder ao SMS para confirmar que viu o alerta. Se não responder, outro contacto é acionado.

⚠️ Problemas Comuns e Soluções

SMS não é enviado

Verifica:

  • Se a conta IFTTT está ativa e com créditos (se aplicável)
  • Se o número de telefone está correto (incluindo indicativo do país: +351 para Portugal)
  • Se o dispositivo está corretamente ligado ao IFTTT
  • Se o applet está ativo (ON)

Demasiados SMS (falsos alarmes)

Solução:

  • Adiciona um delay no IFTTT (ex: só enviar SMS se o alarme durar mais de 10 segundos)
  • Configura no SmartLife um atraso antes de ativar o alarme
  • Cria um "modo silêncio" para quando estás em casa

Custo dos SMS

Solução: Usa alternativas gratuitas:

  • Telegram: Cria um bot e um grupo privado
  • WhatsApp: Existe integração com IFTTT (limitada)
  • Notificação push: App IFTTT no telemóvel

Números errados ou desatualizados

Solução: Revê os contactos periodicamente. Cria um lembrete mensal para verificar.

Sem internet, SMS não funciona?

Solução: O SMS é enviado pelo IFTTT através da internet. Se a internet cair, o SMS não é enviado. Considera um sistema de backup com um router 4G.
