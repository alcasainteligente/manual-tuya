Códigos Temporários

⭐⭐ Intermédio 📍 Porta de entrada ⏱️ 5 minutos

Gera códigos de acesso com validade limitada (horas, dias ou usos). Ideal para empregadas, visitas, manutenção ou Airbnb.

📦 Equipamentos Necessários

  • 🔐 Fechadura inteligente com gestão de códigos Que suporte múltiplos códigos
  • 📱 App SmartLife Para gestão
  • 📅 IFTTT (opcional) Para integração com calendário

💡 Nota: Verifica se a tua fechadura suporta códigos temporários com expiração automática.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Aceder à gestão de códigos

No app, abre a fechadura e procura por "Gestão de Códigos" ou "User Codes".

Passo 2: Criar código temporário manual

Clica em "Adicionar Código".

Escolhe o tipo: "Temporário" ou "Agendado".

Define:

  • Código: 4-8 dígitos (ex: 1234 para a empregada)
  • Nome: "Empregada Maria"
  • Validade: Escolhe datas/horas
  • Ex: todas as quartas-feiras, das 9h às 12h
  • Ou: apenas hoje, das 14h às 18h

Passo 3: Criar código de uso único

Algumas fechaduras permitem códigos que expiram após o primeiro uso.

Ideal para:

  • Receber uma encomenda quando não estás em casa
  • Deixar entrar um técnico uma única vez

Passo 4: Configurar notificações

Ativa notificações para:

  • Código temporário usado (sabes quando a pessoa entrou)
  • Código expirado (se ainda tentarem usar)

Passo 5: Integração com IFTTT (avançado)

Com IFTTT, podes:

  • IF: Evento no Google Calendar "Empregada"
  • THEN: Ativar código temporário para esse dia/hora

Ou:

  • IF: Código temporário usado
  • THEN: Registar numa planilha (data, hora, código)

Passo 6: Apagar código expirado

A fechadura deve apagar automaticamente os códigos expirados. Verifica se a função está ativa.

💡 Dicas e Variações

🧹 Variação 1: Para empregada

Cria um código que funciona apenas:

  • Às quartas-feiras, das 9h às 12h
  • Recebes notificação quando ela entra e sai

🏠 Variação 2: Para Airbnb

Para cada hóspede, gera um código único que expira no dia do checkout.

Podes enviar o código por mensagem antes da chegada.

📦 Variação 3: Para receber encomendas

Cria um código de uso único, válido apenas hoje das 14h às 18h.

Envia ao estafeta e depois o código expira.

👨‍🔧 Variação 4: Para técnicos

Código válido apenas durante a visita agendada.

Podes ver a hora exata em que o técnico entrou e saiu.

📅 Variação 5: Integração com calendário

Usa IFTTT para criar códigos automaticamente baseado em eventos do teu calendário.

⚠️ Problemas Comuns e Soluções

Código não expira automaticamente

Solução: Verifica se a fechadura suporta expiração automática. Alguns modelos precisam que apagues manualmente. Cria um lembrete no telemóvel.

Pessoa chega fora do horário

Solução: Podes abrir remotamente pelo app se estiveres disponível. Ou cria uma janela de tolerância (ex: código funciona 30 minutos antes/depois).

Esqueci de apagar código antigo

Solução: Revê a lista de códigos regularmente. Alguns apps mostram a data de expiração.

Código partilhado com muitas pessoas

Solução: Em vez de um código para todos, cria códigos individuais. Assim sabes quem entrou.

Fechadura não suporta códigos temporários

Solução: Podes simular com automações:

  • Cria um código normal (ex: 9999)
  • Automação: se código 9999 for usado, apagar esse código após 5 minutos
  • Não é perfeito, mas funciona
