Alerta Bateria Fraca

⭐ Iniciante 📍 Geral ⏱️ 4 minutos

Notifica quando a bateria de qualquer sensor está fraca. Evita falhas no sistema de segurança por falta de energia e garante que todos os dispositivos funcionam corretamente.

📦 Equipamentos Necessários

  • 🔋 Sensores com bateria Movimento, abertura, temperatura, etc.
  • 📱 App SmartLife Para configuração
  • 🪛 Pilhas de reserva CR2032, AA, AAA

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Verificar quais sensores reportam bateria

No app SmartLife, abre cada sensor e procura por uma indicação de bateria (percentagem ou ícone).

Nem todos os sensores reportam bateria - apenas os que funcionam a pilhas.

Passo 2: Criar automação para cada sensor

Vai a "Automation" → "+".

IF: "Device Status" → Seleciona o sensor → "Battery level < 15%"

THEN: "Notification" → "🔋 Bateria fraca no sensor [nome do sensor]"

Nome: "Alerta Bateria - [nome do sensor]"

Passo 3: Criar uma automação única (se o app permitir)

Alguns apps permitem criar uma automação que monitoriza todos os dispositivos:

IF: "Any device" → "Battery level < 15%"

THEN: Notificação com o nome do dispositivo (se dinâmico)

Se não for possível, terás de criar uma automação por sensor.

Passo 4: Configurar notificações no telemóvel

Garante que as notificações do app SmartLife estão ativas no teu telemóvel.

Assim recebes os alertas mesmo com o app fechado.

Passo 5: Criar lembrete periódico (opcional)

Podes criar uma automação mensal para verificar manualmente as baterias:

IF: "Timing" → "Every month" no dia 1

THEN: Notificação: "📅 Verificar baterias dos sensores este mês"

💡 Dicas e Variações

📊 Variação 1: Registo de baterias

Cria uma tabela com as datas de troca de cada sensor:

  • Sensor Porta Entrada: trocada em 01/01/2026
  • Sensor Movimento Sala: trocada em 15/01/2026

Assim sabes quando esperar a próxima troca (geralmente 6-12 meses).

🔋 Variação 2: Pilhas de qualidade

Usa pilhas de lítio de marcas reconhecidas (Duracell, Energizer). Duram mais e têm melhor desempenho em temperaturas extremas.

🔄 Variação 3: Troca em massa

Em vez de trocar uma a uma, troca todas as pilhas dos sensores uma vez por ano, numa data fixa (ex: aniversário da casa).

📱 Variação 4: IFTTT para registo

Com IFTTT, podes registar cada alerta de bateria fraca numa planilha Google Sheets, criando um histórico.

⚠️ Variação 5: Alerta crítico

Para sensores críticos (fumo, gás), podes definir um limite mais alto, como 20%, para ter mais tempo de reação.

⚠️ Problemas Comuns e Soluções

Não recebo notificações de bateria fraca

Verifica:

  • Se o sensor realmente reporta nível de bateria (alguns só avisam quando estão muito fracos)
  • Se as notificações do app estão ativas no telemóvel
  • Se a automação está ativa (ON)

Bateria dura menos do que o esperado

Solução:

  • Uso frequente do sensor gasta mais bateria
  • Temperaturas extremas (frio/gelo) reduzem a vida útil
  • Mau contacto ou sensor defeituoso

Sensor desliga sem aviso

Solução: Alguns sensores não avisam antes de desligar. Para esses, define uma troca preventiva a cada 6 meses.

Muitos alertas ao mesmo tempo

Solução: Se vários sensores ficarem com bateria fraca ao mesmo tempo, podem ter sido instalados na mesma data. Programa uma troca geral anual.
