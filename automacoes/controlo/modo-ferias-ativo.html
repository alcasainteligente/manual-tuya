Modo Férias Ativo

⭐⭐⭐ Avançado 🏖️ Férias ⏱️ 8-10 minutos 🏠 Simulação de presença

Quando estás de férias, a casa simula presença ligando e desligando luzes em padrões aleatórios, abrindo e fechando estores em horários variados. Segurança inteligente para viajares descansado.

📦 Equipamentos Necessários

  • 💡 Lâmpadas inteligentes Em várias divisões
  • 🪟 Estores inteligentes Opcional
  • 🔌 Tomadas inteligentes Para simular TV, rádio
  • 📱 App SmartLife Para configuração

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Criar interruptor virtual "Modo Férias"

Vai a "Smart" → "Scenes" → criar um botão virtual.

Nome: "🏖️ Modo Férias"

Este botão vai ativar/desativar todas as simulações.

Passo 2: Criar simulação de manhã (com variação)

Cria múltiplas automações com horários ligeiramente diferentes:

  • Automação A1: IF: 07:30 + Modo Férias = ON → Luz sala = ON 30%
  • Automação A2: IF: 08:15 + Modo Férias = ON → Luz sala = ON 30%
  • Automação A3: IF: 09:00 + Modo Férias = ON → Luz sala = ON 30%

Passo 3: Simulação de tarde

  • Automação T1: IF: 12:30 + Modo Férias = ON → Luz cozinha = ON 40%
  • Automação T2: IF: 13:15 + Modo Férias = ON → Luz cozinha = ON 40%
  • Automação T3: IF: 14:00 + Modo Férias = ON → Luz cozinha = ON 40%

Passo 4: Simulação de noite

  • Automação N1: IF: 19:30 + Modo Férias = ON → Luz sala = ON 50%
  • Automação N2: IF: 20:15 + Modo Férias = ON → Luz sala = ON 50%
  • Automação N3: IF: 21:00 + Modo Férias = ON → Luz sala = ON 50%

Passo 5: Desligar luzes

Cria também automações para desligar em horários variados:

  • Desligar 1: IF: 23:00 + Modo Férias = ON → Luzes = OFF
  • Desligar 2: IF: 23:45 + Modo Férias = ON → Luzes = OFF
  • Desligar 3: IF: 00:30 + Modo Férias = ON → Luzes = OFF

Passo 6: Simular TV (opcional)

  • TV: Ligar entre 20h-23h em horários variados (consumo simulado com tomada)
  • Podes usar uma tomada com uma lâmpada para simular o brilho da TV

Passo 7: Estores (opcional)

  • Manhã: Abrir estores entre 07h-09h
  • Noite: Fechar estores entre 20h-22h

💡 Dicas e Variações

🏠 Variação 1: Padrão de fim de semana

Ao sábado e domingo, os horários podem ser diferentes (mais tarde). Cria automações específicas para esses dias.

🎚️ Variação 2: Intensidade variável

Em vez de ligar sempre à mesma intensidade, varia entre 30% e 70% para parecer mais natural.

🏡 Variação 3: Diferentes divisões

Distribui as luzes por várias divisões:

  • Sala: mais usada à noite
  • Cozinha: mais usada de dia
  • Quartos: menos usados (só se tiveres visitas simuladas)

📊 Variação 4: Relatório diário

Recebe uma notificação diária: "Simulação de presença ativa. Nenhum movimento detetado."

🔊 Variação 5: Som ambiente

Se tiveres colunas inteligentes, podes tocar rádio ou música ambiente em volume baixo em alguns períodos.

⚠️ Problemas Comuns e Soluções

Muitas automações (difícil gerir)

Solução: Usa uma só automação com múltiplos passos e delays, ou usa um serviço como IFTTT com randomização.

Padrão torna-se previsível

Solução: Quanto mais variações tiveres, menos previsível. Cria 5-6 horários diferentes para cada período.

Esqueceste de desativar quando chegas

Solução: Usa geofencing. Quando entrares na zona de casa, desativa automaticamente o Modo Férias.

Luzes ligadas durante o dia (desperdício)

Solução: Usa sensores de luz. Se houver luz natural suficiente, não liga as luzes.
