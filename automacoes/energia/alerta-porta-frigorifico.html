Alerta Porta Frigorífico

⭐ Iniciante 🍳 Cozinha ⏱️ 3-5 minutos ❄️ Evita estragar comida

Recebes uma notificação se a porta do frigorífico ficar aberta mais de 2 minutos. Adeus à conta de luz alta e comida estragada por descuido.

📦 Equipamentos Necessários

  • 🚪 Sensor de porta magnético Para detetar aberto/fechado
  • 🌡️ Sensor de temperatura (opcional) Para monitorizar o interior
  • 📱 App SmartLife Para configuração
  • 🔔 Altifalante inteligente (opcional) Para alerta sonoro

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar sensor de porta

Coloca o sensor magnético na porta do frigorífico.

Testa se deteta a abertura (LED ou app).

Adiciona o sensor ao app SmartLife.

Passo 2: Criar automação básica

Vai a "Automation" → "+".

IF: "Device" → "Sensor Porta Frigorífico" → "Abrir"

THEN: "Delay" → 2 minutos

THEN: "IF" novamente → Verificar se porta ainda está aberta

THEN: "Notification" → "🚨 Porta do frigorífico aberta há mais de 2 minutos!"

Nome: "Alerta Porta Frigorífico"

Passo 3: Versão com alerta sonoro

Além da notificação, podes fazer som na cozinha:

THEN: "Speaker" → "Alexa, diz: Fecha a porta do frigorífico!"

Ou se tiveres uma campainha inteligente, podes fazer bip.

Passo 4: Versão com sensor de temperatura

Se tiveres sensor dentro do frigorífico:

IF: Temperatura interior > 8°C (perigoso para alimentos)

THEN: Notificação urgente + alerta sonoro

Podes verificar se a porta ficou aberta ou se o frigorífico avariou.

Passo 5: Criar exceções para descongelar

Quando estás a descongelar o frigorífico, não queres alertas:

IF: "Manual" → Criar um interruptor virtual "Modo Descongelar"

Quando ativo, desativa a automação de alerta.

💡 Dicas e Variações

💰 Variação 1: Poupança energética

Um frigorífico com porta aberta gasta muito mais:

  • Compressor trabalha sem parar
  • Pode gastar +50% de energia enquanto aberto
  • Comida pode estragar em 2h

📊 Variação 2: Estatísticas de uso

Com sensor, podes ver:

  • Quantas vezes abres o frigorífico por dia
  • Horas de maior uso
  • Se alguém deixa a porta aberta frequentemente

👶 Variação 3: Alerta para crianças

Se tens crianças que adoram abrir o frigorífico:

  • Alerta após 30 segundos (não 2 minutos)
  • Mensagem: "Fechar a porta, por favor!"

🌡️ Variação 4: Congelador

Aplica o mesmo sensor no congelador:

  • Alerta após 1 minuto (perde frio mais rápido)
  • Temperatura ideal: -18°C

📱 Variação 5: Notificação para todos

Se vives com mais pessoas:

  • Envia notificação para todos os telemóveis da casa
  • Quem estiver mais perto pode fechar

⚠️ Problemas Comuns e Soluções

Alerta dispara quando estás a arrumar compras

Causa: A porta fica aberta vários minutos enquanto arrumas.

Solução: Aumenta para 5 minutos ou cria modo "Compras" que desativa temporariamente.

Sensor descola com o calor/ humidade

Solução: Usa fita dupla face 3M ou parafusos. Alguns sensores vêm com íman próprio.

Bateria do sensor dura pouco

Solução: Escolhe sensores Zigbee (duram mais que WiFi). Troca baterias a cada 6 meses.

Porta do frigorífico não fecha bem (borracha gasta)

Solução: O alerta é sintoma, não causa. Verifica a borracha de vedação.

Falso alerta se abrires e fechares rápido

Solução: O delay de 2 minutos já evita isso. Só alerta se ficar mesmo aberto.
