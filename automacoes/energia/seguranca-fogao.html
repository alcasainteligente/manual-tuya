Segurança Fogão

⭐⭐⭐ Avançado 🍳 Cozinha ⏱️ 8-10 minutos 🔥 Prevenção de incêndios

O fogão desliga automaticamente se não houver movimento na cozinha durante 30 minutos ou se detetar fumo. Segurança para idosos, crianças ou para quem vive sozinho.

📦 Equipamentos Necessários

  • 🔥 Fogão inteligente Ou relé para desligar
  • 🚶 Sensor de movimento Na cozinha
  • 💨 Sensor de fumo/calor Próximo ao fogão
  • 🌡️ Sensor de temperatura Para detetar sobreaquecimento
  • 🔌 Tomada inteligente (se for elétrico) Para cortar energia
  • 📱 App SmartLife Para configuração

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar sensores

Coloca o sensor de movimento num local que cubra toda a cozinha.

Coloca o sensor de fumo perto do fogão (não por cima da chama).

Coloca o sensor de temperatura na parede próxima.

Adiciona todos ao app SmartLife.

Nota: Para fogões a gás, precisas de uma válvula solenoide inteligente (instalação por profissional).

Passo 2: Automação por ausência de movimento

IF: "Device" → "Sensor Movimento Cozinha" → "Sem movimento" por 30 minutos

AND: "Device" → "Fogão" = ON (ou consumo > 100W)

THEN: "Fogão" = OFF + Notificação: "⚠️ Fogão desligado por segurança (sem movimento)"

Nome: "Segurança Fogão - Ausência"

Passo 3: Automação por fumo

IF: "Device" → "Sensor Fumo" → "Fumo detetado"

THEN: "Fogão" = OFF (prioridade máxima) + Alarme sonoro + Notificação urgente

Nome: "Segurança Fogão - Fumo"

Passo 4: Automação por temperatura excessiva

IF: "Device" → "Sensor Temperatura Cozinha" → "Above" → 60°C (anormal)

AND: "Fogão" = ON

THEN: "Fogão" = OFF + Notificação: "🔥 Temperatura anormal na cozinha"

Passo 5: Versão para fogão a gás

Precisas de:

  • Válvula solenoide inteligente (corta gás)
  • Sensor de chama (opcional)

IF: Ausência 30 min + Fogão ligado → Fechar válvula gás

⚡ Segurança extra: Sensor de monóxido de carbono (obrigatório para gás).

Passo 6: Notificações para familiares

Se vives sozinho ou tens idosos em casa:

THEN: Enviar notificação para telemóveis dos filhos/cuidadores

"Atenção: Fogão desligado por segurança na casa do avô"

💡 Dicas e Variações

👵 Variação 1: Para idosos

Pessoas mais velhas podem esquecer o fogão ligado:

  • Reduz tempo para 15 minutos (em vez de 30)
  • Adiciona alerta sonoro antes de desligar
  • Envia notificação para cuidador

👶 Variação 2: Segurança crianças

Bloqueio do fogão:

  • Sensor na gaveta dos fósforos/isqueiros (se abrir, notificação)
  • Fogão só liga se adulto na cozinha
  • Desliga automaticamente após 10 min se criança detetada sozinha

📱 Variação 3: Controlo remoto

Podes desligar o fogão de qualquer lugar:

  • App no telemóvel
  • Comando de voz: "Ok Google, desliga o fogão"
  • Botão de pânico no frigorífico (se disponível)

⏰ Variação 4: Cozinhados longos

Para cozinhados que demoram horas (ex: cozido):

  • Cria modo "Cozinhado longo" que desativa a automação por 4h
  • Mas mantém sensor de fumo ativo (segurança)

🏠 Variação 5: Integração com deteção de saída

Se o geofencing detetar que todos saíram de casa:

  • Verificar se fogão está ligado
  • Se sim, desligar automaticamente e notificar

🧯 Variação 6: Extintor automático

Para casas T0 ou muito pequenas:

  • Se fumo detetado E temperatura > 80°C
  • Ativar extintor automático (sistema profissional)

⚠️ Problemas Comuns e Soluções

Fogão desliga enquanto cozinhas (estás parado)

Causa: Sensor de movimento não te deteta se estiveres parado.

Solução: Usa sensor de vibração no fogão (deteta panelas a ferver) ou aumenta tempo para 45 minutos.

Fogão a gás - como desligar automaticamente?

Solução: Válvula solenoide com corta-gás automático. Instalação obrigatória por técnico credenciado. Não faças DIY com gás!

Falso alarme de fumo com grelhados

Solução: O sensor de fumo é para incêndio, não para fumo de cozinha. Usa sensor de calor em vez de fumo, ou posiciona o sensor mais longe.

Cozinha muito grande, sensor não cobre tudo

Solução: Usa 2 sensores de movimento ou um sensor de ângulo alargado.

Fogão elétrico (220V) não pode ser controlado por tomada normal

Solução: Precisa de relé de potência (contactora) instalado no quadro elétrico. Serviço profissional.

Desliga enquanto estás a fazer conservas (horas)

Solução: Cria um botão "Modo Conserva" que desativa a automação por 3h.
