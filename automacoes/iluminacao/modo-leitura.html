Modo Leitura

⭐ Iniciante 📍 Sala / Quarto ⏱️ 4 minutos

Ajusta a iluminação para uma leitura confortável: candeeiro de leitura a 80%, restantes luzes a 20% para criar um ambiente acolhedor sem cansar a vista.

📦 Equipamentos Necessários

  • 💡 Candeeiro de leitura Com lâmpada regulável
  • 💡 Lâmpadas reguláveis Para as restantes luzes
  • 🎮 Botão inteligente ou comando Para ativar o modo
  • 📱 App SmartLife Para configuração

💡 Nota: Para este modo é essencial que as lâmpadas sejam reguláveis (dimmer). Lâmpadas normais só ligam/desligam.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Agrupar as luzes

Para facilitar, cria dois grupos:

  • Vai a "Smart" → "Create Group"
  • Grupo 1: "Candeeiro Leitura" (só o candeeiro)
  • Grupo 2: "Luzes Ambiente" (todas as outras luzes da sala)

Passo 2: Criar cena "Modo Leitura"

Vai a "Smart" → "Scenes" → "Create Scene".

Dá o nome: "Modo Leitura".

Ações da cena (THEN):

  • Candeeiro Leitura: ON, Brightness 80%, Temperatura 4000K (neutro)
  • Luzes Ambiente: ON, Brightness 20%
  • TV/Som (opcional): OFF (para não distrair)

Guarda a cena.

📌 A temperatura 4000K é ideal para leitura - nem muito amarelo, nem muito azul.

Passo 3: Criar cena "Fim de Leitura" (opcional)

Para quando terminaste de ler:

  • Candeeiro Leitura: OFF
  • Luzes Ambiente: 50% (voltar ao normal)

Passo 4: Ativar por botão (recomendado)

Se tiveres um botão inteligente (ex: Botão Tuya, Flic):

  • Vai a "Automation" → "+"
  • IF: Botão = "Click único"
  • THEN: "Execute Scene" → "Modo Leitura"
  • Guarda com nome: "Ativar Leitura por Botão"

Passo 5: Ativar por voz

Se usas Alexa ou Google Assistant:

  • A cena aparece automaticamente nos assistentes
  • Diz: "Alexa, liga Modo Leitura"
  • Ou: "Hey Google, ativar modo leitura"

Passo 6: Ativar por horário (ex: todas as noites)

Se lês sempre à mesma hora:

  • Vai a "Automation" → "+"
  • IF: Timing → "Every day" às 22:00
  • THEN: "Execute Scene" → "Modo Leitura"

💡 Dicas para uma Leitura Perfeita

📐 Posição do candeeiro

O candeeiro de leitura deve estar:

  • Por cima do ombro (para destros, ombro esquerdo)
  • A uma altura que a luz incida diretamente no livro/kindle
  • Sem criar sombras na página

🌡️ Temperatura de cor ideal

  • 2700K-3000K (amarelo): Muito relaxante, mas pode dar sono
  • 4000K (neutro): Ideal para leitura prolongada
  • 5000K+ (frio): Bom para estudo, mas cansa mais à noite

Recomendação: 4000K para leitura noturna, 5000K para estudo durante o dia.

📱 Leitura em tablet/Kindle

Se lês em dispositivos eletrónicos:

  • Reduz a intensidade para 60% (menos reflexo)
  • Usa temperatura mais quente (3000K) para não cansar
  • Podes criar uma cena separada "Leitura Digital"

☕ Com bebida

Se gostas de ler com uma bebida ao lado:

  • Adiciona uma luz muito suave na mesa de apoio (5%)
  • Assim não entornas no escuro

⏰ Temporizador de leitura

Cria uma automação que desliga após 45 minutos (técnica Pomodoro):

  • Na cena "Modo Leitura", adiciona um delay de 45 min
  • Depois do delay, ativa a cena "Fim de Leitura"
  • Assim não adormeces com as luzes ligadas

👓 Para quem usa óculos

Se usas óculos, evita luz por cima da cabeça que pode criar reflexos nas lentes. Prefere luz vinda de lado ou de trás.

⚠️ Problemas Comuns e Soluções

Candeeiro não regula para 80%

Solução: Verifica se a lâmpada é regulável (dimmer). Lâmpadas LED normais não regulam. Procura lâmpadas com símbolo de regulação (ícone de círculo com traço).

Luz muito forte para leitura

Solução: Experimenta 60% em vez de 80%. O ideal é confortável para os olhos, não precisa de ser máximo.

Ambiente muito escuro

Solução: Aumenta as luzes ambiente para 30% ou 40%. O contraste entre o candeeiro (80%) e o resto (20%) pode ser demasiado para algumas pessoas.

Luz do candeeiro reflete no ecrã

Solução: Se lês num tablet, ajusta a posição do candeeiro para que a luz não bata diretamente no ecrã. Luz indireta é melhor para ecrãs.

Adormeço sempre a ler

Solução: Usa temperatura mais fria (5000K) para te manteres acordado. Ou lê durante o dia em vez de à noite.
