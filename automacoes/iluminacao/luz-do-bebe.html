Luz do Bebé

⭐⭐ Intermédio 📍 Quarto Bebé ⏱️ 6 minutos

Acende uma luz suave vermelha (5%) no quarto do bebé quando ele chora durante a noite. Iluminação suficiente para veres o bebé sem o acordar completamente.

📦 Equipamentos Necessários

  • 🔊 Sensor de som Para detetar choro
  • 💡 Lâmpada RGB regulável Para luz vermelha suave
  • 🎤 Monitor de bebé (opcional) Com integração IFTTT
  • 📱 App SmartLife Para configuração

💡 Nota: O SmartLife não tem sensor de som nativo. Podes usar um monitor de bebé com IFTTT ou um sensor de vibração na cama.

📱 Configuração Passo-a-Passo

Passo 1: Escolher o método de deteção

Opção A - Sensor de vibração (mais simples):

  • Coloca um sensor de movimento/vibração por baixo do colchão
  • Deteta quando o bebé se mexe/chora

Opção B - Monitor de bebé com IFTTT:

  • Alguns monitores (ex: Nanit, Eufy) têm integração IFTTT
  • Cria um applet: "Se detetar choro, ligar luz SmartLife"

Passo 2: Instalar a luz no quarto

Coloca uma lâmpada regulável num local que ilumine o bebé mas não o ofusque:

  • Um candeeiro de canto virado para a parede
  • Uma fita LED atrás da cómoda
  • Luz indireta é sempre melhor

Passo 3: Configurar no SmartLife (com sensor de vibração)

Vai a "Automation" → "+".

IF:

  • "Device Status" → Sensor Vibração = "Vibração detetada"
  • AND: Horário entre 21:00 e 08:00

THEN:

  • "Device Control" → Luz Bebé = ON
  • Brightness: 5%
  • Cor: Vermelho escuro (RGB: 139, 0, 0)
  • Delay 10 minutos
  • Luz Bebé = OFF

Nome: "Luz Bebé Noturna"

Passo 4: Configurar no IFTTT (com monitor de bebé)

No IFTTT:

  • IF: "Monitor Bebé" → "Choro detetado"
  • THEN: "SmartLife" → "Turn on with color"
  • Seleciona a lâmpada, cor vermelha, 5%

Cria outro applet para desligar após 10 minutos.

Passo 5: Versão com notificação para os pais

Adiciona uma notificação no telemóvel:

  • No THEN, adiciona "Notification"
  • Mensagem: "👶 O bebé acordou!"

💡 Dicas para o Quarto do Bebé

🔴 Porquê vermelho?

A luz vermelha não inibe a produção de melatonina (hormona do sono) nem no bebé nem nos pais. Assim, depois de acudires o bebé, voltam a adormecer mais facilmente.

🌡️ Temperatura do quarto

Se tiveres sensor de temperatura, adiciona uma condição extra: só acender se temperatura estiver normal (20-22°). Se estiver muito quente/frio, recebes alerta diferente.

🎵 Som branco

Se tiveres um altifalante, podes programar para tocar som branco quando o bebé acorda, para ajudar a acalmar.

⏱️ Tempo de delay

10 minutos é suficiente para veres o bebé e ele voltar a dormir. Se for para mamar, podes criar uma versão com 20-30 minutos.

⚠️ Problemas Comuns e Soluções

Sensor de vibração deteta movimentos dos pais

Solução: Coloca o sensor apenas num canto do colchão ou usa um sensor de som em vez de vibração.

Luz muito forte

Solução: 5% pode ser muito em alguns quartos pequenos. Experimenta 3% ou 2%. Ou coloca a luz mais afastada.

Bebé acorda mas não chora logo

Solução: Usa sensor de movimento na cama. Quando o bebé começa a mexer-se, a luz acende antes mesmo de chorar.
