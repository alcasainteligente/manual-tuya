Luz Dispensa

⭐ Iniciante 📍 Cozinha ⏱️ 4 minutos

Acende a luz da dispensa automaticamente quando abres a porta. Adeus a procurar coisas no escuro!

📦 Equipamentos Necessários

  • 🔌 Sensor de porta magnético Tuya Zigbee ou Wi-Fi
  • 💡 Lâmpada ou fita LED Para interior da dispensa
  • 🔋 Baterias Para o sensor

💡 Nota: Se a dispensa não tiver eletricidade, podes usar uma fita LED a bateria. Mas terás de a carregar ocasionalmente.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar o sensor na porta da dispensa

Coloca o sensor de porta:

  • Parte maior (sensor) na ombreira da porta
  • Parte menor (íman) na porta
  • Certifica-te que ficam alinhados quando a porta está fechada
📌 A distância entre o sensor e o íman não deve ultrapassar 1cm quando a porta está fechada.

Passo 2: Instalar a luz na dispensa

Coloca a luz no teto da dispensa ou numa prateleira superior:

  • Deve iluminar bem todas as prateleiras
  • Se usares fita LED, cola ao longo do teto ou das laterais

Passo 3: Configurar os dispositivos no app

Adiciona o sensor e a luz ao app SmartLife.

Dá nomes:

  • Sensor: "Sensor Porta Dispensa"
  • Luz: "Luz Dispensa"

Passo 4: Criar automação para acender

Vai a "Automation" → "+".

IF: "Device Status" → Sensor Porta Dispensa = "Abrir"

THEN: "Device Control" → Luz Dispensa = ON

Nome: "Luz Dispensa ao Abrir"

Passo 5: Criar automação para desligar

IF: Sensor Porta Dispensa = "Fechar"

THEN:

  • Delay 1 minuto (tempo para tirares o que precisas)
  • Luz Dispensa = OFF

Nome: "Desligar Luz Dispensa"

Passo 6: Versão alternativa (mais simples)

Se preferires uma única automação:

IF: Sensor = "Abrir"

THEN:

  • Luz = ON
  • Delay 2 minutos
  • Luz = OFF

Nota: Esta versão desliga sempre após 2 minutos, mesmo que a porta ainda esteja aberta.

💡 Dicas para uma Dispensa Perfeita

⏱️ Tempo ideal

1-2 minutos é suficiente para a maioria das idas à dispensa. Se costumas demorar mais a organizar, aumenta para 3 minutos.

🔋 Sensor com bateria

Os sensores de porta duram cerca de 1 ano com bateria. Quando a bateria estiver fraca, o app avisa. Substitui por pilhas de qualidade (Duracell, Energizer).

💡 Luz LED fria (6000K)

Para a dispensa, o ideal é luz fria (6000K) porque ajuda a ver melhor as etiquetas e prazos de validade. Luz amarela pode distorcer as cores.

📦 Múltiplas prateleiras

Se a dispensa for funda, considera colocar uma fita LED em cada prateleira (comandada pelo mesmo interruptor). Assim toda a dispensa fica iluminada.

⚠️ Problemas Comuns e Soluções

Sensor não deteta a abertura

Solução: Verifica o alinhamento do íman com o sensor. Pode ter-se descolado. A distância máxima é 1cm. Se for maior, o sensor não deteta.

Luz não desliga

Solução: Se usaste a versão com sensor de fecho, verifica se o sensor está a detetar que a porta fechou. Se a porta não fechar bem, o sensor pode continuar a detetar "aberto".

Bateria do sensor dura pouco

Solução: Se abres a dispensa muitas vezes, é normal gastar mais bateria. Usa pilhas de lítio (duram mais) ou considera um sensor com fio se for muito usado.
