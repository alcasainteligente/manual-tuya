Proteção Livros / Biblioteca

⭐⭐ Intermédio 📍 Biblioteca/Escritório ⏱️ 5 minutos

Mantém a humidade entre 45% e 55% em bibliotecas ou locais com livros, documentos ou instrumentos musicais. Previne o aparecimento de bolor, páginas onduladas e danos em papéis valiosos.

📦 Equipamentos Necessários

  • 💧 Sensor de humidade Para a biblioteca
  • 🌀 Desumidificador inteligente Para reduzir humidade
  • 💨 Humidificador inteligente (opcional) Para aumentar humidade se necessário
  • 📱 App SmartLife Para configuração

💡 Nota: Livros e documentos devem ser mantidos entre 45% e 55% de humidade. Acima disso, há risco de bolor; abaixo, as páginas ficam quebradiças.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar sensor na estante/biblioteca

Coloca o sensor de humidade numa prateleira, entre os livros, para medir a humidade real a que os livros estão expostos.

Passo 2: Configurar desumidificador

Se a humidade for alta, vamos ligar o desumidificador.

Adiciona o desumidificador ao app SmartLife.

Passo 3: Criar automação para humidade alta

Vai a "Automation" → "+".

IF: Sensor Humidade > 55%

THEN: Desumidificador = ON

Nome: "Desumidificador Biblioteca ON"

Passo 4: Criar automação para desligar desumidificador

IF: Sensor Humidade < 50%

THEN: Desumidificador = OFF

Nome: "Desumidificador Biblioteca OFF"

Passo 5: Se a humidade for baixa (opcional)

Em climas muito secos, podes precisar de humidificador:

IF: Sensor Humidade < 45%

THEN: Humidificador = ON

IF: Sensor Humidade > 50%

THEN: Humidificador = OFF

Passo 6: Alerta de risco

IF: Sensor Humidade > 60% por mais de 2 horas

THEN: Notificação: "⚠️ Humidade crítica na biblioteca! Risco de bolor nos livros."

💡 Dicas e Variações

📚 Variação 1: Livros raros ou antigos

Para livros de valor, considera uma vitrine fechada com controlo de humidade independente. Podes colocar um sensor dentro da vitrine.

🎻 Variação 2: Instrumentos musicais

Instrumentos de madeira (guitarras, pianos, violinos) também sofrem com humidade. Aplica a mesma automação para a sala onde estão guardados.

📄 Variação 3: Documentos importantes

Para arquivos de documentos, mantém a humidade entre 40-50%. Considera também um sensor de temperatura para evitar extremos.

📊 Variação 4: Registo histórico

Com IFTTT, regista a humidade ao longo do tempo. Se houver um problema, consegues provar as condições a que os livros foram expostos.

🏠 Variação 5: Circulação de ar

Além do controlo de humidade, garante que há circulação de ar na estante. Podes usar pequenas ventoinhas se a estante for fechada.

⚠️ Problemas Comuns e Soluções

Humidade muito alta apesar do desumidificador

Solução: O desumidificador pode ser pequeno para o espaço. Calcula a capacidade necessária (litros/dia) para o tamanho da divisão. Pode ser necessário um segundo aparelho.

Desumidificador desliga sozinho (depósito cheio)

Solução: Alguns modelos têm saída para mangueira. Se não tiveres, programa para desligar após X horas e receber notificação para esvaziar.

Humidade baixa no inverno

Solução: O aquecimento central seca o ar. Usa o humidificador automático para compensar.

Mofo já existente

Solução: Limpa os livros afetados com pano seco e areja a divisão. Depois aplica a automação para evitar que volte.
