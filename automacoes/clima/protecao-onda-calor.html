Proteção Onda Calor

⭐⭐ Intermédio 🪟 Todas as divisões ⏱️ 6-8 minutos 🌡️ Até 5°C mais fresco

Em dias de calor extremo, os estores fecham estrategicamente para impedir que o sol aqueça o interior. A casa mantém-se fresca sem precisar de AC ligado o dia todo.

📦 Equipamentos Necessários

  • 🪟 Motorização para estores Todas as fachadas
  • 🌡️ Sensor de temperatura exterior Para detetar onda de calor
  • 🌡️ Sensor de temperatura interior Para medir eficácia
  • ☀️ Sensor de luminosidade (opcional) Para versão otimizada
  • 📱 App SmartLife Para configuração

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Configurar os motores dos estores

Adiciona todos os motores de estore ao app SmartLife.

Agrupa por fachada: "Estores Sul", "Estores Nascente", "Estores Poente", "Estores Norte".

Isto é importante porque cada fachada recebe sol em horas diferentes.

Passo 2: Versão básica (fecha tudo)

Vai a "Automation" → "+".

IF: "Weather" → "Temperature" → "Above" → 32°C (temperatura de alerta)

THEN: "Device Control" → Grupo "Todos Estores" = Fechar 0%

Nome: "Onda Calor - Fechar Tudo"

Nota: 32°C é um bom limiar para Portugal continental. Ajusta conforme a região.

Passo 3: Versão inteligente (por fachada/horário)

Fachada Nascente (sol da manhã):

IF: Temperatura > 30°C + Horário 08:00-12:00 → Fechar estores nascente

Fachada Sul (sol todo o dia):

IF: Temperatura > 30°C + Horário 10:00-18:00 → Fechar estores sul

Fachada Poente (sol da tarde - o pior):

IF: Temperatura > 28°C + Horário 14:00-20:00 → Fechar estores poente

Fachada Norte (pouco sol):

IF: Temperatura > 35°C (calor extremo) → Fechar estores norte

Passo 4: Versão com sensor de temperatura interior

Se a casa já estiver fresca, podes não precisar de fechar:

IF: Temperatura exterior > 32°C

AND: Temperatura interior < 24°C (casa ainda fresca)

THEN: Fechar estores apenas parcialmente (50%) para manter alguma luz

IF: Temperatura exterior > 32°C

AND: Temperatura interior > 26°C (já está a aquecer)

THEN: Fechar estores completamente (0%)

Passo 5: Versão com sensor de luminosidade

Para dias muito nublados, mesmo com calor, podes não precisar de fechar:

IF: Temperatura > 30°C

AND: Sensor luminosidade > 5000 lux (sol forte)

THEN: Fechar estores

IF: Temperatura > 30°C

AND: Sensor luminosidade < 2000 lux (nublado)

THEN: Manter estores abertos ou fechar apenas parcialmente

Passo 6: Criar notificações de alerta

IF: "Weather" → "Temperature" → "Above" → 35°C

THEN: "Notification" → Enviar mensagem: "🔥 Alerta Onda Calor! Estores fechados automaticamente. Temperatura interior: XX°C"

Inclui um link rápido para abrir estores se preferires.

💡 Dicas e Variações

🌡️ Variação 1: Combinar com AC

Cria uma estratégia completa para ondas de calor:

  • Nível 1 (30-32°C): Fechar estores das fachadas mais expostas
  • Nível 2 (33-35°C): Fechar todos os estores + ligar ventiladores
  • Nível 3 (>35°C): Fechar tudo + ligar AC em modo económico (24°C)

🌙 Variação 2: Aproveitar a noite

À noite, quando a temperatura exterior desce, podes abrir tudo para arejar:

IF: Horário 22:00-06:00 + Temperatura exterior < 22°C

THEN: Abrir estores e janelas (se tiveres abertura automática)

Isto arrefece a casa naturalmente para o dia seguinte.

🏠 Variação 3: Proteção para divisões específicas

Algumas divisões sofrem mais com o calor:

  • Sótão/último andar: Fechar estores mais cedo (a partir de 28°C)
  • Quartos virados a poente: Fechar a partir das 14h
  • Cozinha: Fechar sempre que temperatura > 30°C (já esquenta com fogão)

📊 Variação 4: Estatísticas de eficácia

Com sensores interiores e exteriores, podes medir o ganho:

  • Sem proteção: interior chega a 35°C quando exterior está a 38°C
  • Com proteção: interior mantém-se a 28°C
  • Diferença: até 7°C mais fresco, poupança de 50% no AC

🌍 Variação 5: Proteção para plantas

Se tiveres plantas que precisam de luz, podes criar exceções:

  • Manter estores da sala com plantas abertos 30% para luz indireta
  • Fechar apenas a partir das 14h (hora de sol mais forte)

🤖 Variação 6: Integração com previsão meteorológica

Usa a previsão para antecipar a onda de calor:

IF: Previsão de temperatura máxima > 35°C para amanhã

THEN: Preparar casa: fechar estores mais cedo, avisar para carregar baterias, etc.

⚠️ Problemas Comuns e Soluções

Casa fica escura como uma caverna

Causa: Fechaste todos os estores em todas as fachadas.

Solução: Mantém os estores da fachada norte abertos (não recebe sol direto) ou abre parcialmente (30%) os estores nas fachadas que já não estão a receber sol naquele momento.

Estores fecham e abrem várias vezes

Causa: Temperatura oscila perto do limiar (ex: 31-32°C).

Solução: Adiciona histerese. Usa 32°C para fechar e 28°C para abrir, ou usa um temporizador (ex: manter fechado por pelo menos 2 horas).

Onda de calor à noite

Causa: Noites tropicais com temperatura > 25°C.

Solução: Cria automação noturna: manter estores fechados durante a noite (para não entrar calor) e ligar AC ou ventilador.

Estores fechados mas casa continua quente

Causa: O calor já estava acumulado nas paredes antes de fechares.

Solução: Fecha os estores mais cedo (antes de a casa aquecer). Para ondas de calor, fecha logo de manhã, antes de saíres para o trabalho.

Onda de calor dura vários dias

Causa: Acumulação de calor dia após dia.

Solução: À noite, quando a temperatura exterior baixar, abre tudo para arejar. Se não baixar (noite tropical), mantém fechado e usa AC.

Estores fecham e as tuas plantas morrem

Causa: Plantas precisam de luz.

Solução: Cria uma exceção para as divisões com plantas. Fecha apenas parcialmente (50%) ou usa estores com lâminas reguláveis para filtrar a luz.

Não tens sensores de temperatura

Solução: Usa a temperatura da tua estação meteorológica local (weather API no app SmartLife) ou cria automações baseadas apenas no horário (ex: fechar das 12h às 18h em julho/agosto).
