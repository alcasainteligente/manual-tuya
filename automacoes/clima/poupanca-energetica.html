Poupança Energética

⭐⭐⭐ Avançado 📍 Toda a casa ⏱️ 10 minutos

Ajusta a temperatura do AC/aquecimento para usar mais intensamente nos horários de tarifa mais barata, reduzindo o consumo nas horas mais caras. Pode reduzir a conta de eletricidade em 20-30%.

📦 Equipamentos Necessários

  • 🌡️ AC ou aquecimento inteligente Compatível com Tuya
  • 📱 App SmartLife Para configuração
  • 📊 Conhecimento do tarifário Horas cheias/vazio
  • 🔌 Tomadas inteligentes (opcional) Para outros equipamentos

💡 Nota: Em Portugal, a tarifa bi-horária tem horas de vazio (mais barato) e horas cheias (mais caro). Geralmente, vazio: 22h-8h dias úteis e todo o fim de semana.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Identificar os horários do teu tarifário

Consulta a tua fatura de eletricidade para saber os horários de vazio e cheia.

Exemplo típico em Portugal continental:

  • Horas vazio (mais barato): 22h-8h dias úteis + fim de semana todo
  • Horas cheias (mais caro): 8h-22h dias úteis

Passo 2: Criar automação para horas vazio (conforto máximo)

Vai a "Automation" → "+".

IF:

  • "Timing" → "Repeat" → Dias úteis, das 22:00 às 08:00
  • E todo o fim de semana

THEN:

  • AC = ON, temperatura 21°C (inverno) / 23°C (verão) - temperatura de conforto

Nome: "Modo Conforto Horas Vazio"

Passo 3: Criar automação para horas cheias (poupança)

IF:

  • "Timing" → "Repeat" → Dias úteis, das 08:00 às 22:00

THEN:

  • AC = ON, temperatura 19°C (inverno) / 25°C (verão) - 2°C abaixo/acima do conforto para poupar

Nome: "Modo Poupança Horas Cheias"

Passo 4: Criar automação para desligar quando não há ninguém

Para poupança máxima, combina com sensor de presença:

IF: Sensor Movimento = "Sem movimento" por 30 minutos

AND: Horário entre 08:00 e 22:00

THEN: AC = OFF

Passo 5: Criar automação para tomadas (opcional)

Podes aplicar o mesmo princípio a outros equipamentos:

IF: Horário vazio

THEN:

  • Ligar máquina de lavar roupa (se tiveres programação)
  • Ligar esquentador elétrico
  • Ligar carregador de carro elétrico

Passo 6: Versão com sensor de temperatura exterior

Para maior eficiência, podes ajustar a temperatura conforme a temperatura exterior:

  • Se exterior < 10°C: aquecimento a 21°C nas horas vazio, 19°C nas horas cheias
  • Se exterior entre 10-20°C: aquecimento a 19°C nas horas vazio, 17°C nas horas cheias
  • Se exterior > 20°C: desligar aquecimento

Passo 7: Criar resumo de poupança (opcional)

Com IFTTT, podes registar o consumo e calcular a poupança estimada.

💡 Dicas e Variações

📊 Variação 1: Tarifário tri-horário

Alguns tarifários têm três períodos: ponta, cheia e vazio. Ajusta as automações conforme:

  • Ponta (mais caro): 18h-22h - temperatura 18°C (inverno) / 26°C (verão)
  • Cheia: 8h-18h - temperatura 19°C / 25°C
  • Vazio (mais barato): 22h-8h + fins de semana - temperatura 21°C / 23°C

🔋 Variação 2: Painéis solares

Se tiveres painéis solares, programa os equipamentos para funcionar durante as horas de produção (dia).

📱 Variação 3: Notificação de poupança

Recebe um relatório mensal com a estimativa de poupança: "Este mês poupaste 15€ com as automações de clima."

🏠 Variação 4: Zonas diferentes

Podes ter temperaturas diferentes em divisões diferentes, conforme o uso:

  • Sala: temperatura de conforto durante o dia
  • Quartos: temperatura mais baixa durante o dia, conforto à noite

🌡️ Variação 5: Pré-aquecimento/arrefecimento

Antes da hora de ponta (mais cara), podes pré-aquecer/arrefecer a casa para que o sistema possa desligar durante as horas caras.

⚠️ Problemas Comuns e Soluções

Mudança de hora (inverno/verão)

Solução: As automações baseadas em horas fixas precisam de ser ajustadas quando muda a hora. O SmartLife não ajusta automaticamente para o horário de verão/inverno. Terás de atualizar manualmente.

Tarifário diferente ao fim de semana

Solução: Cria automações separadas para dias úteis e fim de semana. No fim de semana, como é tudo vazio, podes ter conforto todo o dia.

AC não atinge a temperatura desejada nas horas vazio

Solução: Podes precisar de ligar o AC mais cedo. Ex: em vez de ligar às 22h, liga às 21h30 para que às 22h já esteja à temperatura desejada.

Muitas automações, confusão

Solução: Cria cenas "Inverno" e "Verão" que ativam as automações corretas para cada estação.

Esquecer de mudar entre inverno/verão

Solução: Podes usar a temperatura exterior como gatilho. Se temperatura exterior > 20°C durante 3 dias consecutivos, muda para modo verão.
