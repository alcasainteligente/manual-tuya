Modo Férias Clima

⭐⭐ Intermédio 📍 Toda a casa ⏱️ 8 minutos

Mantém a temperatura da casa entre 15°C e 30°C durante as férias. Evita canos congelados no inverno, calor extremo no verão, e previne o aparecimento de bolor e mofo.

📦 Equipamentos Necessários

  • 🌡️ Sensor de temperatura Pelo menos um central
  • 🔥❄️ AC ou aquecimento inteligente Para controlar temperatura
  • 💧 Sensor de humidade (opcional) Para prevenir bolor
  • 📱 App SmartLife Para configuração

💡 Nota: Durante as férias, o objetivo não é ter conforto, é proteger a casa. Temperaturas muito baixas ou muito altas podem causar danos.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Criar cena "Modo Férias Clima ON"

Vai a "Smart" → "Scenes" → "Create Scene".

Dá o nome: "Modo Férias Clima - Ativar".

Ações: (não precisa de ações, é apenas um "disparador" para ativar as automações)

Passo 2: Criar automação para evitar frio extremo

Vai a "Automation" → "+".

IF:

  • "Device Status" → Sensor Temperatura < 5°C
  • AND: Modo Férias está ativo (podes usar uma variável ou simplesmente ativar/desativar a automação manualmente)

THEN:

  • Aquecimento = ON, temperatura 15°C
  • Notificação: "❄️ Temperatura muito baixa! Aquecimento ligado para proteger a casa."

Nome: "Proteção Férias Frio"

Passo 3: Criar automação para evitar calor extremo

IF: Sensor Temperatura > 35°C

THEN:

  • AC = ON, temperatura 30°C
  • Notificação: "🔥 Temperatura muito alta! AC ligado para proteger a casa."

Nome: "Proteção Férias Calor"

Passo 4: Criar automação para desligar

IF: Sensor Temperatura entre 10°C e 30°C (zona segura)

THEN:

  • Aquecimento = OFF
  • AC = OFF

Nome: "Desligar Clima Férias"

Passo 5: Criar automação para controlo de humidade (opcional)

Para evitar bolor, a humidade não deve ser muito alta:

IF: Sensor Humidade > 70%

THEN:

  • Ventilação/Desumidificador = ON
  • Notificação: "💧 Humidade elevada. Desumidificador ligado."

Passo 6: Criar cena "Modo Férias Clima OFF"

Para quando voltares:

Nome: "Modo Férias Clima - Desativar"

Ações: Desativar as automações anteriores (ou simplesmente desligar manualmente).

Passo 7: Ativação automática por geofencing (opcional)

Podes criar uma automação que ative o modo férias quando todos saem de casa por mais de 24h.

💡 Dicas e Variações

❄️ Variação 1: Inverno rigoroso

Se o inverno for muito frio na tua zona (abaixo de -10°C exterior), pode ser necessário um limite mais alto:

  • Ligar aquecimento a 8°C em vez de 5°C
  • Manter temperatura mínima de 18°C em zonas com canos expostos

☀️ Variação 2: Verão extremo

Em zonas de muito calor (ex: Alentejo), podes precisar de:

  • Ligar AC a 32°C em vez de 35°C
  • Fechar estores durante o dia para manter o interior fresco

📱 Variação 3: Alerta para vizinho

Se a temperatura sair dos limites e o sistema não conseguir corrigir, envia um alerta para um vizinho de confiança que tenha chave de casa.

🏠 Variação 4: Zonas diferenciadas

Podes ter limites diferentes para diferentes zonas:

  • Cave (canos): mínimo 10°C
  • Sótão (telhado): máximo 40°C
  • Zonas habitáveis: 15-30°C

💰 Variação 5: Poupança máxima

Se quiseres poupar ao máximo, podes desligar tudo e confiar apenas nos alertas. Mas corres o risco de danos se algo falhar.

⚠️ Problemas Comuns e Soluções

Falha de energia durante a ausência

Solução:

  • Se a energia cair, o sistema não funciona. Considera um gerador ou UPS para o router e termostato.
  • Em último caso, no inverno, deixa as torneiras a pingar ligeiramente (água em movimento não congela tão facilmente).
  • Pede a um vizinho para verificar a casa periodicamente.

Internet falha, não recebes alertas

Solução: Alguns sistemas têm backup 4G. Se não tiveres, combina com um vizinho que verifique pessoalmente.

AC/aquecimento avaria durante a ausência

Solução: Se receberes alertas de temperatura e não conseguires resolver à distância, contacta um vizinho ou serviço de assistência.

Limites muito apertados disparam constantemente

Solução: Ajusta os limites com margem. Ex: em vez de 5-35°C, usa 8-32°C. Evita que o sistema ligue/desligue constantemente.

Humidade elevada durante as férias

Solução: Se não tiveres desumidificador, deixa armários abertos e portas interiores abertas para circulação de ar.
