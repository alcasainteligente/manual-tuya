Aquecimento Zonado

⭐⭐⭐ Avançado 📍 Toda a casa ⏱️ 10 minutos

Aquece apenas as divisões ocupadas da casa, poupando energia. Sensores de movimento detetam onde estás e ligam o aquecimento só nesses locais. Ideal para famílias e para reduzir a fatura de energia.

📦 Equipamentos Necessários

  • 🔌 Sensores de movimento Um por divisão (ou mais)
  • 🌡️ Aquecedores inteligentes Ou termostatos por zona
  • 🌡️ Sensores de temperatura Para feedback e evitar sobreaquecimento
  • 📡 Hub Zigbee (se necessário) Para sensores

💡 Nota: Podes usar aquecedores a óleo, radiadores elétricos ou bombas de calor, desde que tenham controlo Tuya.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Mapear as divisões da casa

Identifica as principais divisões que vais aquecer:

  • Sala
  • Quarto principal
  • Quartos das crianças
  • Escritório
  • Cozinha (pode não precisar, pois já aquece com eletrodomésticos)

Passo 2: Instalar sensores de movimento

Coloca um sensor de movimento em cada divisão, apontado para a zona de passagem.

No quarto, evita apontar para a cama para não detetar movimento durante o sono.

Passo 3: Instalar sensores de temperatura

Coloca um sensor de temperatura em cada divisão, longe de fontes de calor (aquecedores, janelas).

Passo 4: Criar grupos por divisão

Vai a "Smart" → "Create Group" e cria grupos:

  • "Aquecimento Sala" - aquecedor da sala + sensor temperatura sala
  • "Aquecimento Quarto" - aquecedor do quarto + sensor temperatura quarto
  • "Aquecimento Escritório" - etc.

Passo 5: Criar automação para cada divisão (ligar)

Vai a "Automation" → "+".

IF:

  • Sensor Movimento Sala = "Movimento detetado"
  • AND: Sensor Temperatura Sala < 19°C (temperatura desejada)

THEN: Aquecimento Sala = ON, temperatura 21°C

Nome: "Aquecimento Sala ON"

Repete para cada divisão.

Passo 6: Criar automação para desligar

IF: Sensor Movimento Sala = "Sem movimento" por 30 minutos

THEN: Aquecimento Sala = OFF

Nome: "Aquecimento Sala OFF"

Passo 7: Criar automação de temperatura máxima (segurança)

IF: Sensor Temperatura Sala > 23°C

THEN: Aquecimento Sala = OFF (evitar sobreaquecimento)

Passo 8: Criar modo noturno (opcional)

Durante a noite, podes querer aquecer apenas os quartos:

  • IF: Horário entre 23:00 e 07:00
  • THEN: Desligar aquecimento da sala (mesmo com movimento)

💡 Dicas e Variações

🏠 Variação 1: Prioridade de divisões

Se o sistema de aquecimento não conseguir aquecer tudo ao mesmo tempo, define prioridades:

  • Sala tem prioridade máxima (onde a família se reúne)
  • Quartos têm prioridade média
  • Escritório tem prioridade baixa

👨‍👩‍👧 Variação 2: Deteção por pessoa

Com pulseiras ou telemóveis, podes detetar quem está em cada divisão e ajustar a temperatura conforme a preferência de cada um.

⏱️ Variação 3: Pré-aquecimento

Se souberes que vais para uma divisão a uma certa hora (ex: escritório às 9h), programa para começar a aquecer 30 minutos antes.

📊 Variação 4: Poupança comprovada

Estima-se que aquecimento zonado pode poupar 20-30% na fatura de energia. Cria um registo para comparar consumos.

🌡️ Variação 5: Temperaturas diferentes por divisão

Cada pessoa pode ter preferências diferentes:

  • Sala: 21°C
  • Quarto principal: 19°C
  • Quarto de bebé: 22°C

⚠️ Problemas Comuns e Soluções

Aquecedor não desliga quando sais da divisão

Solução: Ajusta o tempo de delay. 30 minutos pode ser muito se a divisão for usada por pouco tempo. Experimenta 15 minutos.

Sensor não deteta movimento (pessoa parada)

Solução: Sensores PIR detetam movimento, não presença. Se estiveres parado a ler ou a ver TV, o sensor pode não detetar. Usa um sensor de presença (microondas) ou combina com sensor de porta.

Divisão arrefece demasiado rápido

Solução: Se a divisão for mal isolada, o aquecimento pode não ser eficiente. Melhora o isolamento (janelas, portas) ou usa um aquecedor mais potente.

Múltiplas automações (uma por divisão) confusas

Solução: Cria uma cena "Gestão Aquecimento" que ativa todas as automações de uma vez. Assim é mais fácil ativar/desativar o sistema completo.

Crianças deixam o aquecimento ligado no quarto vazio

Solução: Reduz o tempo de delay para 5-10 minutos nos quartos das crianças. Ou usa um sensor de porta na entrada do quarto.
