Cozinha sem Fumos

⭐⭐ Intermédio 📍 Cozinha ⏱️ 5 minutos

Liga o exaustor automaticamente quando deteta fumo, vapor ou aumento de temperatura no fogão. Mantém o ar da cozinha limpo e evita que os odores se espalhem pela casa.

📦 Equipamentos Necessários

  • 🌫️ Sensor de fumo/partículas Para detetar fumos de cozinha
  • 🌡️ Sensor de temperatura (opcional) Para detetar calor do fogão
  • 🌀 Exaustor inteligente Ou tomada inteligente + exaustor normal
  • 📱 App SmartLife Para configuração

💡 Nota: O exaustor deve ter potência suficiente para o tamanho da cozinha. Para cozinhas americanas, é ainda mais importante.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Instalar sensor de fumo/partículas

Coloca o sensor perto do fogão, mas não diretamente sobre a chama. Ideal: na exaustor ou no teto próximo.

Certifica-te que não é um detetor de incêndio (que seria demasiado sensível).

Passo 2: Configurar o exaustor

Se o exaustor não for inteligente, liga-o a uma tomada inteligente.

Adiciona o dispositivo ao app SmartLife.

Dá nomes: "Sensor Fumo Cozinha", "Exaustor Cozinha".

Passo 3: Criar automação para ligar (fumo/partículas)

Vai a "Automation" → "+".

IF: Sensor Partículas > 50 µg/m³ (ou outro limiar)

THEN: Exaustor Cozinha = ON (velocidade alta)

Nome: "Exaustor Fumo ON"

Passo 4: Criar automação para temperatura (opcional)

IF: Sensor Temperatura Cozinha > 40°C (fogão ligado)

THEN: Exaustor Cozinha = ON (velocidade média)

Nome: "Exaustor Calor ON"

Passo 5: Criar automação para desligar

IF: Sensor Partículas < 30 µg/m³ E Sensor Temperatura < 35°C por 10 minutos

THEN: Exaustor Cozinha = OFF

Nome: "Exaustor OFF"

Passo 6: Versão com timer (alternativa)

IF: Sensor Partículas > 50

THEN: Exaustor = ON, Delay 30 minutos, Exaustor = OFF

💡 Dicas e Variações

🍳 Variação 1: Combinação com fogão inteligente

Se tiveres um fogão inteligente, podes ligar o exaustor sempre que o fogão for ligado.

💨 Variação 2: Velocidade variável

Conforme a quantidade de fumo, podes ter velocidades diferentes:

  • 50-100 µg/m³: velocidade baixa
  • 100-200 µg/m³: velocidade média
  • >200 µg/m³: velocidade alta (algo queimou!)

🔇 Variação 3: Modo silencioso

À noite, se cozinhares, podes querer um exaustor mais silencioso. Programa velocidade baixa entre as 22h e as 8h.

🏠 Variação 4: Cozinha americana

Em cozinhas abertas para a sala, o exaustor é ainda mais importante. Podes ligá-lo sempre que houver atividade na cozinha (sensor de movimento + temperatura).

🔔 Variação 5: Alerta de excesso

Se o sensor detetar níveis muito altos (algo a queimar), recebe uma notificação urgente.

⚠️ Problemas Comuns e Soluções

Exaustor liga com vapor de água (não fumo)

Solução: Aumenta o limiar de partículas. Vapor de água tem partículas maiores mas menos densas. Experimenta 80 µg/m³ em vez de 50.

Exaustor não desliga

Solução: O sensor pode estar a ler partículas residuais. Usa o timer em vez de sensor para desligar.

Sensor falso com comida queimada

Solução: Acontece! O exaustor vai ligar na mesma, que é o que queres. Depois areja a casa.

Exaustor fraco para a cozinha

Solução: Verifica a capacidade (m³/h). Cozinhas grandes podem precisar de um exaustor mais potente ou de dois pontos de extração.
