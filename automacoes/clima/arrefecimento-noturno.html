Arrefecimento Noturno

⭐ Iniciante 📍 Quarto ⏱️ 5 minutos

Liga o ventilador ou AC se a temperatura do quarto subir acima de 24° durante a noite. Garante uma noite de sono fresca sem teres de acordar para ajustar.

📦 Equipamentos Necessários

  • 🌡️ Sensor de temperatura Para o quarto
  • 🌀 Ventilador inteligente ou AC Compatível com Tuya
  • 📱 App SmartLife Para configuração

💡 Nota: O sensor deve ser colocado longe do AC/ventilador para não ser influenciado diretamente.

📱 Configuração Passo-a-Passo no App SmartLife

Passo 1: Colocar sensor de temperatura no quarto

Coloca o sensor num local que não seja afetado diretamente pelo AC/ventilador para leitura correta da temperatura ambiente.

Ideal: numa mesa de cabeceira ou parede oposta ao AC.

Passo 2: Adicionar dispositivos ao app

Adiciona o sensor de temperatura e o ventilador/AC ao app SmartLife.

Dá nomes: "Sensor Temperatura Quarto", "Ventilador Quarto".

Passo 3: Criar automação para ligar

Vai a "Automation" → "+".

IF:

  • "Device Status" → Sensor Temperatura > 24°C
  • AND: Horário entre 23:00 e 07:00

THEN: "Device Control" → Ventilador Quarto = ON, velocidade média

(Se for AC: temperatura 22°C, modo cool)

Nome: "Arrefecimento Noturno ON"

Passo 4: Criar automação para desligar (por temperatura)

IF: Sensor Temperatura < 22°C

THEN: Ventilador Quarto = OFF

Nome: "Arrefecimento Noturno OFF"

Passo 5: Versão com timer (alternativa)

Se preferires desligar após um tempo fixo:

Na automação de ligar, adiciona:

  • Delay 4 horas
  • Ventilador = OFF

Passo 6: Ajuste fino (histerese)

Para evitar que o ventilador ligue/desligue constantemente:

  • Liga a 24°C
  • Desliga a 22°C (margem de 2°)

💡 Dicas e Variações

🌬️ Variação 1: Ventilador vs AC

Podes criar duas automações diferentes:

  • 24°C - 26°C: Ligar ventilador (poupança)
  • > 26°C: Ligar AC (mais potente)

⏱️ Variação 2: Desligar após adormecer

Muitas pessoas preferem o ventilador apenas para adormecer. Programa para desligar após 2 horas (delay na automação).

🌡️ Variação 3: Temperatura adaptativa

Em noites muito quentes, podes querer uma temperatura mais baixa. Cria diferentes limiares para diferentes temperaturas exteriores (se tiveres sensor exterior).

👶 Variação 4: Quarto de bebé

Para bebés, os limites devem ser mais rigorosos:

  • Ligar ventilador a 23°C (nunca AC direto)
  • Nunca abaixo de 20°C

🔊 Variação 5: Som branco

Se o ventilador fizer barulho, podes usá-lo como "som branco" para ajudar a dormir. Nesse caso, não desligues durante a noite.

⚠️ Problemas Comuns e Soluções

Ventilador/AC liga e desliga constantemente

Solução: Ajusta os limites com uma margem (histerese). Ex: Liga a 24°, desliga a 22°. Isto evita ciclos muito curtos.

Sensor de temperatura lê valores errados

Verifica:

  • Se o sensor está perto do AC/ventilador (lê temperatura mais baixa)
  • Se está perto de uma janela (correntes de ar)
  • Se está exposto a luz solar direta durante o dia

Não arrefece o suficiente

Solução: Pode ser necessário mais tempo. Em vez de desligar por temperatura, mantém o AC ligado durante a noite toda a 22°C.

Acordo com frio

Solução: Ajusta a temperatura de desligar para 23°C em vez de 22°C. O corpo arrefece durante a noite, podes precisar de menos frio.

Ventilador faz muito barulho

Solução: Escolhe um modelo silencioso ou usa velocidade baixa durante a noite. Podes programar para velocidade alta apenas no início.
